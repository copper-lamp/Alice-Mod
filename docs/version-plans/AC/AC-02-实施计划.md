## 第1章 概述

### 1.1 总览

Agent Core 模块共需完成 **15 个版本**迭代，覆盖 **10 个子系统**的开发和交付。采用**2 人并行开发**模式，开发者 A 和开发者 B 按各自负责方向同步推进。

### 1.2 版本节奏

| 阶段 | 版本 | 周期 | 说明 |
|------|:----:|:----:|------|
| 基础建设 | V1-V3 | 1 周/版本 | 项目骨架、TCP 通信、工作区管理 |
| 执行核心 | V4-V6 | 1-2 周/版本 | Function Calling、提示词系统、LLM 调度 |
| 工具矩阵 | V7-V9 | 1-2 周/版本 | UI 面板、日志系统 |
| 智能增强 | V10-V13 | 1-2 周/版本 | QQ 机器人、记忆系统、任务系统 |
| 集成发布 | V14-V15 | 2 周/版本 | 事件触发器、功能冻结、打包发布 |

**总时长**：约 20 周（5 个月）

### 1.3 依赖关系总图

```
V1 ──> V2 ──> V3 ──> V4 ──> V5 ──> V6 ──> V7 ──> V8 ──> V9 ──> V10 ──> V11 ──> V12 ──> V13 ──> V14 ──> V15
       │      │      │      │      │      │      │      │      │       │       │       │       │       │
```
