## 第5章 关键依赖关系

### 5.1 外部依赖

| 依赖 | 版本 | 说明 |
|------|------|------|
| Node.js | 20.x | AC 运行时 |
| JDK | 21 | JE 编译与运行 |
| Minecraft Server | 1.21.4 | MC 服务端 jar |
| Fabric Loader | 对应版本 | Mod 加载器 |
| Carpet Mod | ≥ 1.4.161 | 假人生成 |

### 5.2 内部依赖

| 模块 | 说明 |
|------|------|
| `packages/agent-core` | AC 端，TCP 服务器、工具调度、提示词组装 |
| `packages/adapter-java` | JE 端，Fabric Mod、工具实现、假人管理 |
| `serverjava/` | MC 服务端运行目录，包含 mods 配置 |

### 5.3 启动顺序

```
1. AC 服务器（TCP 27541）必须先启动
2. JE MC 服务器（fabric-server-launch.jar）后启动
3. JE 模组启动后自动连接 AC，握手 → 注册工具 → 状态上报
4. AC 收到工具注册后，即可向 JE 发送工具调用
```
