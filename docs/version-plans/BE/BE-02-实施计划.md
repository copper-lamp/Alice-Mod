## 第1章 概述

### 1.1 项目总览

Adapter Core BE 是基于 LeviLamina 插件 + Node.js/TypeScript + BDS API 的基岩版接入核心模块，共 15 个版本迭代，实现 26 个工具。总预估工时 **20 周（5 个月）**。

### 1.2 版本节奏

| 阶段 | 版本 | 周期 | 工时 |
|:----:|:----:|:----:|:----:|
| 基础建设 | V1-V3 | 第 1-3 周 | 3 周 |
| 执行核心 | V4-V6 | 第 4-8 周 | 5 周 |
| 工具矩阵 | V7-V9 | 第 9-11 周 | 3 周 |
| 智能增强 | V10-V13 | 第 12-17 周 | 6 周 |
| 集成发布 | V14-V15 | 第 18-20 周 | 3 周 |

### 1.3 总依赖关系

```
V1 (插件骨架) ──> V2 (TCP客户端) ──> V3 (工具注册+状态上报)
                                              │
                                              ▼
                                       V4 (移动系统) ←── 所有工具的基础
                                    ┌─────┼─────┐
                                    ▼     ▼     ▼
                              V5(背包) V6(生存+方块) V7(生物+战斗)
                                    │     │     │
                                    ▼     ▼     ▼
                              V8(感知) ── V9(对话) ── V10(假人管理)
                                    │     │     │
                                    ▼     ▼     ▼
                              V11(数据持久化) ── V12(配置接入)
                                    │     │
                                    ▼     ▼
                              V13(任务对接) ── V14(性能优化+稳定性)
                                          │
                                          ▼
                                     V15(打包发布)
```

**断线重连逻辑**：

```typescript
const RECONNECT_INTERVALS = [1000, 2000, 4000, 8000, 16000];
const MAX_RECONNECT_ATTEMPTS = 5;

private scheduleReconnect(): void {
  if (this.reconnectAttempts >= MAX_RECONNECT_ATTEMPTS) {
    logger.error('已达最大重连次数，停止重连');
    return;
  }
  const delay = RECONNECT_INTERVALS[this.reconnectAttempts];
  logger.info(`将在 ${delay}ms 后重连 (第 ${this.reconnectAttempts + 1} 次)`);
  this.reconnectTimer = setTimeout(() => this.attemptReconnect(), delay);
  this.reconnectAttempts++;
}

private async attemptReconnect(): Promise<void> {
  this.state = ConnectionState.RECONNECTING;
  try {
    await this.connect();
    await this.sendHandshake();
    await this.registerTools();  // 重连后自动重新注册工具
    this.reconnectAttempts = 0;
    logger.info('重连成功');
  } catch (err) {
    this.scheduleReconnect();
  }
}
```

#### 关键实现要点

**A* 3D 核心**：

```typescript
interface PathNode {
  x: number; y: number; z: number;
  g: number;     // 起点到此节点的实际代价
  h: number;     // 启发式估计代价（到终点的曼哈顿距离）
  f: number;     // g + h
  parent: PathNode | null;
}

class Pathfinder {
  private readonly MAX_SEARCH_RANGE = 128;
  private readonly STEP_LENGTH = 1;
  private readonly JUMP_HEIGHT = 1;   // 可自动跨越的高度差
  private readonly MAX_JUMP_HEIGHT = 2; // 可尝试跳跃+搭方块的高度

  async findPath(start: Vec3, end: Vec3, options: PathOptions): Promise<Vec3[]> {
    // 1. 校验范围：起点到终点距离 > 128 时直接失败
    // 2. 初始化 openSet 和 closedSet
    // 3. 主循环（最大迭代 100000 次防止死循环）
    // 4. 邻居扩展：26 方向（3x3x3 除自身）
    // 5. 路径平滑：去除冗余拐点
  }

  private getNeighbors(node: PathNode, world: WorldAccess): PathNode[] {
    // 返回可通行的相邻节点
    // 检查：方块是否可站立（非空气？），头部空间是否足够（2格高）
    // 检查：是否需要跳跃（1格高差自动跳，2格高差尝试搭方块）
    // 检查：是否为水域（切换到游泳模式）
    // 检查：是否为危险方块（岩浆等），绕行
  }

  private heuristic(a: Vec3, b: Vec3): number {
    // 3D 曼哈顿距离 + 高度惩罚
    const dx = Math.abs(a.x - b.x);
    const dy = Math.abs(a.y - b.y) * 2;  // 高度变化权重加倍
    const dz = Math.abs(a.z - b.z);
    return dx + dy + dz;
  }
}
```

**寻路参数**：

| 参数 | 默认值 | 说明 |
|------|:------:|------|
| 步长 | 1 | 每次移动 1 格 |
| 最大搜索范围 | 128 | 超过 128 格返回失败 |
| 可自动跳跃高度 | 1 | 1 格高度差自动跳 |
| 最大尝试高度 | 2 | 2 格高度差尝试搭方块 |
| 游泳检测 | 自动 | 检测到水方块自动切换 |
| 最大迭代次数 | 100000 | 防止死循环 |
| 路径平滑 | 是 | 去除 collinear 中间点 |
