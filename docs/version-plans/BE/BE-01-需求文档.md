## 第1章 概述

### 1.1 模块定位

Adapter Core BE 是 McAgent 三层解耦架构中的**接入核心（基岩版）**，作为运行在 Minecraft Bedrock Dedicated Server（BDS）上的 **LeviLamina 插件 + Node.js/TypeScript** 运行时，是系统的"身体"。负责：

- **执行AI 引擎**：将 Agent Core 下发的工具调用转换为游戏内具体操作（移动、挖掘、放置、战斗等）
- **26 个工具实现**：涵盖感知、移动、背包、生物交互、生存、方块、对话 7 大类
- **TCP 客户端通信**：主动连接 Agent Core，进行握手认证、心跳保活、工具注册与调用
- **状态上报**：实时采集假人状态，每 2s 周期性上报
- **数据持久化**：SQLite 存储配置、记忆元数据和执行日志

### 1.2 技术栈

| 组件 | 技术选型 | 版本要求 | 说明 |
|------|---------|---------|------|
| 游戏服务端 | Minecraft Bedrock Dedicated Server | 1.21.x | 基岩版官方服务端 |
| 插件加载器 | LeviLamina | 0.12.x+ | BDS 插件加载框架 |
| 运行时 | Node.js | 20 LTS+ | TCP 客户端、工具注册、逻辑执行 |

## 第4章 非功能需求

### 4.1 性能指标

| 指标 | 要求 | 测量方式 |
|------|:----:|----------|
| 状态上报周期 | ≤ 2s | 从插件采集到 TCP 消息发出的耗时 |
| 工具执行超时（非移动类） | < 30s | 超时返回 TOOL_TIMEOUT 错误 |
| 工具执行超时（移动类） | < 120s | 超时返回 TOOL_TIMEOUT 错误 |
| 寻路计算耗时（128 范围内） | < 50ms | A* 算法执行时间 |
| 状态上报单次耗时 | < 100ms | 从采集到序列化完成 |
| TPS 影响 | < 5% | 插件开启前后基准测试对比 |
| 内存占用增量 | < 200MB | 相对于裸 BDS 的额外内存 |
| 插件启动时间 | < 5s | 从 LeviLamina 加载到工具注册完成 |
| TCP 重连恢复时间 | ≤ 16s | 断线后到重连工具注册完成 |

### 4.2 稳定性要求

| 要求 | 指标 |
|------|:----:|
| 连续运行时间 | ≥ 72h 无崩溃 |
| 支持假人数量 | ≤ 3 个（同服务器） |
| 插件崩溃恢复 | 不导致 BDS 主进程崩溃 |
| 断线重连成功率 | ≥ 99%（网络正常时） |
| 工具调用成功率 | ≥ 95%（正常游戏条件下） |
| 内存泄漏 | 72h 运行后内存增长 < 50MB |

### 4.3 安全要求

| 要求 | 说明 |
|------|------|
| 认证机制 | TCP 连接需携带 auth_token 握手认证 |
| 命令黑名单 | 配置文件中定义禁止执行的游戏命令 |
| 操作权限 | 假人操作不应导致服务器崩溃或数据丢失 |
| 日志审计 | 所有工具调用记录到 SQLite logs 表 |
| 敏感信息保护 | auth_token 不写入日志文件 |

| TypeScript | 5.x | 严格模式开启 |

### 5.2 开发环境约束

| 项目 | 约束 |
|------|------|
| 操作系统 | Windows Server 2022+（推荐）/ Ubuntu 22.04+ |
| IDE | VS Code（推荐，集成 ESLint + Prettier） |
| 包管理 | pnpm workspaces monorepo |
| 代码规范 | 遵循 [standards/01](../standards/01-代码规范与风格指南.md) |
| 通信协议 | 遵循 [protocols/01](../protocols/01-通信协议规范.md) |

### 5.3 接口约束

| 接口 | 约束 |
|------|------|
| IToolModule | 必须实现 metadata() + execute(params, ctx) |
| 工具注册 | 自动扫描 tools/ 目录，每个工具子目录含 manifest.json + index.js |
| 状态上报 | JSON-RPC 2.0 Notification，不等待响应 |
| 工具调用 | JSON-RPC 2.0 Request/Response，同步执行 |
| 心跳响应 | 收到 ping 后立即回复 pong |

### 5.4 部署约束

| 项目 | 约束 |
|------|------|
| TCP 端口 | 默认 27541（与 Agent Core 通信） |
| 目录结构 | plugins/mcagent-adapter-be/{tools/, config.json, index.js, manifest.json} |
| JSON 入口 | BDS 根目录下生成 mcagent_instance.json |
| 数据库路径 | plugins/mcagent-adapter-be/data/mcagent.db |
