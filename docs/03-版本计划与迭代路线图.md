## 第1章 概述

### 1.1 并行开发策略

项目划分 **3 个并行开发模块**，各模块由不同开发者同时推进，通过版本里程碑对齐集成：

| 模块 | 代号 | 技术栈 | 开发者分配 |
|------|------|--------|-----------|
| **Agent Core** | AC | Electron + TypeScript/Node.js + React | 2 人 |
| **Adapter Core BE** | BE | LeviLamina 插件 + Node.js + BDS API | 2 人 |
| **Adapter Core JE** | JE | Fabric 模组 + Java 21 + Minecraft API | 2 人 |

> **Shared 库**（类型定义/工具 Schema/常量）由 Agent Core 团队在 V1 先行产出，BE/JE 参照定义自行实现等价类型，不独立部署。

### 1.2 工具归属

最终 43 个工具分属不同模块实现：

| 归属模块 | 工具类别 | 工具数 | 说明 |
|----------|---------|:------:|------|
| **Agent Core** | QQ 外部连接 | 2 | 通过 OneBot 协议收发消息/查询信息 |
| **Agent Core** | 学习与记忆 | 8 | SQLite + Chroma 存储与检索 |
| **Agent Core** | 任务与目标 | 7 | 任务创建/调度/分解/管理 |
| **Adapter Core** | 感知系统 | 5 | 环境查询 |
| **Adapter Core** | 移动系统 | 3 | 寻路移动/骑乘 |
| **Adapter Core** | 背包与装备 | 4 | 物品管理 |
| **Adapter Core** | 生物交互 | 4 | 战斗/交互/牵引 |
| **Adapter Core** | 生存与状态 | 3 | 饮食/睡眠/物品使用 |
| **Adapter Core** | 方块操作 | 4 | 挖掘/放置/区域操作 |
| **Adapter Core** | 对话与社交 | 3 | 聊天/私聊/消息管理 |
| **总计** | | **43** | |

## 第2章 版本总览

### 2.1 研发阶段划分

15 个版本划分为 5 个阶段：

```
阶段一：基础建设（V1-V3）→ 项目骨架 + 通信通道
   ↓
阶段二：执行核心（V4-V6）→ 执行AI主体 + Agent Core调度
   ↓
阶段三：工具矩阵（V7-V9）→ 全部43个工具就绪
   ↓
阶段四：智能增强（V10-V12）→ 记忆/任务/QQ/事件
   ↓
阶段五：集成发布（V13-V15）→ 集成测试 + 性能优化 + 正式发布
```
