# 生物交互工具设计（Entity Interaction Tools Design）

> 版本：v1.0
> 日期：2026-07-01
> 架构：LLM 切换状态 → 执行AI 持续执行 → 状态变化时返回

---

## 0. 设计原则

### 0.1 核心概念

**战斗模式由LLM切换，执行AI持续执行，状态变化时才返回**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 切换战斗模式（迎击/防御/被动）                      │
│  - 调用生物交互工具（喂食/交易/牵引等）                 │
└────────────────┬────────────────────────────────────┘
                 │ 切换模式/调用工具
                 ▼
        ┌────────────────┐
        │  战斗模式        │  ← 持续状态
        │  - aggressive   │     执行AI自动执行
        │  - defensive    │     不需要每次攻击返回
        │  - passive      │
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 自动攻击     │
        │  - 自动躲避     │
        │  - 自动走位     │
        │  - 状态变化时   │
        │    返回给LLM    │
        └────────┬───────┘
                 │ 状态变化返回
                 ▼
        ┌────────────────┐
        │  状态变化通知    │
        │  - 目标死亡     │
        │  - 血量过低     │
        │  - 武器损坏     │
        └────────────────┘
```

### 0.2 设计原则

| 原则 | 说明 |
|---|---|
| **状态切换** | LLM切换战斗状态，执行AI持续执行 |
| **减少返回** | 不需要每次攻击都返回，只在状态变化时返回 |
| **生物交互** | 喂食、交易等直接操作 |
| **简单直接** | 工具粒度适中，避免过度设计 |

## 2. 战斗状态设计

### 2.1 战斗状态类型

```typescript
enum CombatMode {
  AGGRESSIVE = 'aggressive',  // 迎击：主动接近并攻击
  DEFENSIVE = 'defensive',    // 防御：保持距离，格挡为主
  PASSIVE = 'passive'         // 被动：不主动攻击，只防御
}
```

### 2.2 状态切换工具

```typescript
interface SetCombatModeParams {
  mode: 'aggressive' | 'defensive' | 'passive';
  target_id?: string;  // 目标实体ID（可选，不填则攻击最近敌人）
}

interface SetCombatModeResult {
  success: boolean;
  reason?: 'success' | 'target_not_found' | 'error';
  currentMode?: string;
  targetId?: string;
  errorMessage?: string;
}
```

### 4.2 执行AI接口

```typescript
// 执行AI接口（生物交互）
interface ExecutionAI {
  // 战斗状态
  setCombatMode(mode: 'aggressive' | 'defensive' | 'passive', targetId?: string): Promise<SetCombatModeResult>;
  stopCombat(): Promise<StopCombatResult>;
  
  // 生物交互
  interactEntity(entityId: string, action: 'feed' | 'breed' | 'trade' | 'tame' | 'shear' | 'milk', tradeIndex?: number): Promise<InteractEntityResult>;
  
  // 牵引操作
  leadEntity(entityId: string, action: 'lead' | 'release'): Promise<LeadEntityResult>;
}
```

### 4.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  private combatState: CombatState | null = null;
  private combatLoop: NodeJS.Timeout | null = null;
  
  constructor(player: Player) {
    this.player = player;
  }
  
  async setCombatMode(mode: 'aggressive' | 'defensive' | 'passive', targetId?: string): Promise<SetCombatModeResult> {
    this.stopCombatLoop();
    
    let target: Entity | null = null;
    if (targetId) {
      target = getEntityById(targetId);
      if (!target) {
        return { success: false, reason: 'target_not_found' };
      }
    } else {
      target = this.findNearestHostile();
      if (!target) {
        return { success: false, reason: 'target_not_found' };
      }
    }
    
    this.combatState = { mode, target, startTime: Date.now() };
    this.startCombatLoop();
    
    return { success: true, reason: 'success', currentMode: mode, targetId: target.uniqueId };
  }
  
  async stopCombat(): Promise<StopCombatResult> {
    if (!this.combatState) {
      return { success: false, reason: 'not_in_combat' };
    }
    this.stopCombatLoop();
    this.combatState = null;
    return { success: true, reason: 'success' };
  }
  
  private startCombatLoop() {
    this.combatLoop = setInterval(() => {
      if (!this.combatState) return;
      
      const { mode, target } = this.combatState;
      
      if (!target || target.getHealth() <= 0) {
        this.notifyLLM({ type: 'target_dead', data: { targetId: target?.uniqueId }, timestamp: Date.now() });
        this.stopCombat();
        return;
      }
      
      if (this.player.getHealth() < 5) {
        this.notifyLLM({ type: 'low_health', data: { currentHealth: this.player.getHealth() }, timestamp: Date.now() });
        this.stopCombat();
        return;
      }
      
      switch (mode) {
        case 'aggressive':
          this.executeAggressiveCombat(target);
          break;
        case 'defensive':
          this.executeDefensiveCombat(target);
          break;
        case 'passive':
          // 不主动攻击，只防御
          break;
      }
    }, 100);
  }
  
  private stopCombatLoop() {
    if (this.combatLoop) {
      clearInterval(this.combatLoop);
      this.combatLoop = null;
    }
  }
  
  private executeAggressiveCombat(target: Entity) {
    const distance = this.calculateDistance(this.player.pos, target.pos);
    if (distance > 3) {
      this.moveTo(target.pos);
    }
    if (distance <= 3) {
      this.player.attack(target);
    }
  }
  
  private executeDefensiveCombat(target: Entity) {
    const distance = this.calculateDistance(this.player.pos, target.pos);
    const keepDistance = 4;
    
    if (distance < keepDistance - 1) {
      this.moveAway(target.pos);
    } else if (distance > keepDistance + 1) {
      this.moveTo(target.pos);
    }
    
    if (distance <= keepDistance) {
      this.player.attack(target);
    }
  }
  
  async interactEntity(entityId: string, action: 'feed' | 'breed' | 'trade' | 'tame' | 'shear' | 'milk', tradeIndex?: number): Promise<InteractEntityResult> {
    const entity = getEntityById(entityId);
    if (!entity) {
      return { success: false, reason: 'not_found' };
    }
    
    const distance = this.calculateDistance(this.player.pos, entity.pos);
    if (distance > 5) {
      return { success: false, reason: 'too_far' };
    }
    
    switch (action) {
      case 'feed': {
        if (!this.isFeedable(entity)) return { success: false, reason: 'not_feedable' };
        const food = this.findFoodForEntity(entity);
        if (!food) return { success: false, reason: 'no_food' };
        this.player.feedEntity(entity, food);
        return { success: true, reason: 'success', entityName: entity.name, details: { foodUsed: food.name } };
      }
      
      case 'breed': {
        if (!this.isBreedable(entity)) return { success: false, reason: 'not_breedable' };
        const food = this.findFoodForEntity(entity);
        if (!food) return { success: false, reason: 'no_food' };
        this.player.breedEntity(entity, food);
        return { success: true, reason: 'success', entityName: entity.name, details: { babySpawned: true } };
      }
      
      case 'trade': {
        if (entity.type !== 'villager') return { success: false, reason: 'not_villager' };
        const trades = this.getVillagerTrades(entity);
        if (tradeIndex === undefined || tradeIndex >= trades.length) return { success: false, reason: 'invalid_trade' };
        const trade = trades[tradeIndex];
        this.player.tradeWithVillager(entity, tradeIndex);
        return {
          success: true, reason: 'success', entityName: entity.name,
          details: { tradeDetails: { bought: trade.sell, sold: trade.buy, times: 1 } }
        };
      }
      
      case 'tame': {
        if (!this.isTameable(entity)) return { success: false, reason: 'not_tameable' };
        const result = this.player.tameEntity(entity);
        return { success: true, reason: 'success', entityName: entity.name, details: { tamed: result.tamed, attempts: result.attempts } };
      }
      
      case 'shear': {
        if (!this.isShearable(entity)) return { success: false, reason: 'not_shearable' };
        const wool = this.player.shearEntity(entity);
        return { success: true, reason: 'success', entityName: entity.name, details: { woolDropped: wool } };
      }
      
      case 'milk': {
        if (!this.isMilkable(entity)) return { success: false, reason: 'not_milkable' };
        this.player.milkEntity(entity);
        return { success: true, reason: 'success', entityName: entity.name, details: { milkObtained: true } };
      }
    }
  }
  
  async leadEntity(entityId: string, action: 'lead' | 'release'): Promise<LeadEntityResult> {
    const entity = getEntityById(entityId);
    if (!entity) {
      return { success: false, reason: 'not_found' };
    }
    
    const distance = this.calculateDistance(this.player.pos, entity.pos);
    if (distance > 5) {
      return { success: false, reason: 'too_far' };
    }
    
    if (action === 'lead') {
      if (!this.isLeashable(entity)) return { success: false, reason: 'not_leashable' };
      this.player.leadEntity(entity);
      return { success: true, reason: 'success', entityName: entity.name };
    } else {
      this.player.releaseEntity(entity);
      return { success: true, reason: 'success', entityName: entity.name };
    }
  }
  
  private calculateDistance(a: Position, b: Position): number {
    return Math.sqrt(Math.pow(a.x - b.x, 2) + Math.pow(a.y - b.y, 2) + Math.pow(a.z - b.z, 2));
  }
}
```

## 6. 总结

### 6.1 核心设计

| 组件 | 职责 |
|---|---|
| **LLM（大脑）** | 切换战斗模式，调用生物交互工具 |
| **战斗模式** | 持续状态，执行AI自动执行 |
| **状态变化通知** | 目标死亡/血量过低时通知LLM |
| **生物交互** | 通过 action 参数区分喂食/繁殖/交易/驯服/剪毛/挤奶 |

### 6.2 关键特性

1. **模式切换**：LLM切换战斗模式（aggressive/defensive/passive）
2. **持续执行**：执行AI自动执行战斗逻辑，不需要每次攻击返回
3. **状态变化通知**：目标死亡/血量过低时主动通知LLM
4. **精简接口**：4 个工具覆盖所有生物交互操作，降低 LLM 决策复杂度

### 6.3 工具列表

| 类别 | 工具 |
|---|---|
| 战斗模式 | set_combat_mode, stop_combat |
| 生物交互 | interact_entity |
| 实体操作 | lead_entity |
