# 生物交互工具设计（Entity Interaction Tools Design）

> 版本：v1.0
> 日期：2026-07-01
> 架构：LLM 切换状态 → 执行AI 持续执行 → 状态变化时返回
> 协议版本：遵循 [00-工具协议规范.md](./00-工具协议规范.md)

---

## 0. 设计原则

### 0.1 核心概念

**战斗模式由LLM切换，执行AI持续执行，状态变化时才返回**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 切换战斗模式（迎击/防御/被动）                      │
│  - 调用生物交互工具（喂食/交易/牵引等）                 │
└────────────────┬────────────────────────────────────┘
                 │ 切换模式/调用工具
                 ▼
        ┌────────────────┐
        │  战斗模式        │  ← 持续状态
        │  - aggressive   │     执行AI自动执行
        │  - defensive    │     不需要每次攻击返回
        │  - passive      │
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 自动攻击     │
        │  - 自动躲避     │
        │  - 自动走位     │
        │  - 状态变化时   │
        │    返回给LLM    │
        └────────┬───────┘
                 │ 状态变化返回
                 ▼
        ┌────────────────┐
        │  状态变化通知    │
        │  - 目标死亡     │
        │  - 血量过低     │
        │  - 武器损坏     │
        └────────────────┘
```

### 0.2 设计原则

| 原则 | 说明 |
|---|---|
| **状态切换** | LLM切换战斗状态，执行AI持续执行 |
| **减少返回** | 不需要每次攻击都返回，只在状态变化时返回 |
| **生物交互** | 喂食、交易等直接操作 |
| **简单直接** | 工具粒度适中，避免过度设计 |

## 2. 战斗状态设计

### 2.1 战斗状态类型

```typescript
enum CombatMode {
  AGGRESSIVE = 'aggressive',  // 迎击：主动接近并攻击
  DEFENSIVE = 'defensive',    // 防御：保持距离，格挡为主
  PASSIVE = 'passive'         // 被动：不主动攻击，只防御
}
```

### 2.2 状态切换工具：`set_combat_mode`

**参考协议**：[ParameterSchema](./00-工具协议规范.md#22-参数规范parameterschema) | [ResultEnvelope](./00-工具协议规范.md#3-返回信封resultenvelope) | [ErrorCode](./00-工具协议规范.md#4-错误码标准errorcode)

**参数定义（ParameterSchema）**：

| 参数 | 类型 | 必填 | 描述 | 枚举值 |
|------|------|:----:|------|--------|
| `mode` | `string` | 是 | 战斗模式 | `aggressive`, `defensive`, `passive` |
| `targetId` | `string` | 否 | 目标实体ID（不填则攻击最近敌人） | — |

**返回定义（ResultEnvelope）**：

```typescript
// 成功时 data 结构
interface SetCombatModeData {
  currentMode: string;   // 当前战斗模式
  targetId?: string;     // 目标实体ID
}

// 完整返回类型
type SetCombatModeResult = ResultEnvelope<SetCombatModeData>;
```

**错误码**：

| 错误码 | 含义 |
|--------|------|
| `TARGET_NOT_FOUND` | 目标实体不存在 |
| `INTERNAL_ERROR` | 内部错误 |

**示例调用**：

```
LLM 调用: set_combat_mode(mode='aggressive', targetId='zombie_123')
返回:
{
  "success": true,
  "data": {
    "currentMode": "aggressive",
    "targetId": "zombie_123"
  },
  "meta": {
    "duration": 500,
    "cost": {}
  }
}
```

**自然语言返回**：

```

```

### 3.1 `interact_entity`（与生物交互）

**参考协议**：[ParameterSchema](./00-工具协议规范.md#22-参数规范parameterschema) | [ResultEnvelope](./00-工具协议规范.md#3-返回信封resultenvelope) | [ErrorCode](./00-工具协议规范.md#4-错误码标准errorcode)

**参数定义（ParameterSchema）**：

| 参数 | 类型 | 必填 | 描述 | 枚举值 |
|------|------|:----:|------|--------|
| `entityId` | `string` | 是 | 实体ID | — |
| `action` | `string` | 是 | 交互动作 | `feed`, `breed`, `trade`, `tame`, `shear`, `milk` |
| `tradeIndex` | `number` | 否 | 交易选项索引，仅在 `trade` 模式下使用 | — |

**返回定义（ResultEnvelope）**：

```typescript
// 成功时 data 结构
interface InteractEntityData {
  entityName?: string;
  details?: {
    foodUsed?: string;
    babySpawned?: boolean;
    tamed?: boolean;
    attempts?: number;
    woolDropped?: number;
    milkObtained?: boolean;
    tradeDetails?: {
      bought: { name: string; count: number };
      sold: { name: string; count: number };
      times: number;
    };
  };
}

// 完整返回类型
type InteractEntityResult = ResultEnvelope<InteractEntityData>;
```

**错误码**：

| 错误码 | 含义 | 触发条件 |
|--------|------|----------|
| `NOT_FOUND` | 实体不存在 | 指定的 entityId 找不到 |
| `TOO_FAR` | 距离太远 | 超出交互距离（5格） |
| `NOT_FOUND` | 实体不可喂食 | 该实体在喂食上下文中不存在 |
| `INVALID_PARAMS` | 参数无效 | 该实体不可繁殖/不是村民/交易选项无效/不可驯服/已驯服/不可剪毛/不可挤奶 |
| `ITEM_NOT_FOUND` | 物品不足 | 背包中无食物/无剪刀/无桶/交易物品不足 |
| `NO_FOOD` | 没有食物 | 背包中无适合该实体的食物 |
| `INTERNAL_ERROR` | 内部错误 | 未预期的运行时错误 |

### 4.2 执行AI接口

```typescript
// 执行AI接口（生物交互）
interface ExecutionAI {
  // 战斗状态
  setCombatMode(mode: 'aggressive' | 'defensive' | 'passive', targetId?: string): Promise<ResultEnvelope<SetCombatModeData>>;
  stopCombat(): Promise<ResultEnvelope<void>>;
  
  // 生物交互
  interactEntity(entityId: string, action: 'feed' | 'breed' | 'trade' | 'tame' | 'shear' | 'milk', tradeIndex?: number): Promise<ResultEnvelope<InteractEntityData>>;
  
  // 牵引操作
  leadEntity(entityId: string, action: 'lead' | 'release'): Promise<ResultEnvelope<LeadEntityData>>;

```

### 4.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  private combatState: CombatState | null = null;
  private combatLoop: NodeJS.Timeout | null = null;
  
  constructor(player: Player) {
    this.player = player;
  }
  
  async setCombatMode(mode: 'aggressive' | 'defensive' | 'passive', targetId?: string): Promise<ResultEnvelope<SetCombatModeData>> {
    this.stopCombatLoop();
    
    let target: Entity | null = null;
    if (targetId) {
      target = getEntityById(targetId);
      if (!target) {
        return {
          success: false,
          error: { code: 'TARGET_NOT_FOUND', message: `目标实体不存在: ${targetId}` },
          meta: { duration: 50 }
        };
      }
    } else {
      target = this.findNearestHostile();
      if (!target) {
        return {
          success: false,
          error: { code: 'TARGET_NOT_FOUND', message: '附近无敌对目标' },
          meta: { duration: 100 }
        };
      }
    }
    
    this.combatState = { mode, target, startTime: Date.now() };
    this.startCombatLoop();
    
    return {
      success: true,
      data: { currentMode: mode, targetId: target.uniqueId },
      meta: { duration: 200 }
    };
  }
  
  async stopCombat(): Promise<ResultEnvelope<void>> {
    if (!this.combatState) {
      return {
        success: false,
        error: { code: 'NOT_FOUND', message: '当前不在战斗中' },
        meta: { duration: 10 }
      };
    }
    this.stopCombatLoop();
    this.combatState = null;
    return {
      success: true,
      data: null,
      meta: { duration: 50 }
    };
  }
  
  private startCombatLoop() {
    this.combatLoop = setInterval(() => {
      if (!this.combatState) return;
      
      const { mode, target } = this.combatState;
      
      if (!target || target.getHealth() <= 0) {
        this.notifyLLM({ type: 'target_dead', data: { targetId: target?.uniqueId }, timestamp: Date.now() });
        this.stopCombat();
        return;
      }
      
      if (this.player.getHealth() < 5) {
        this.notifyLLM({ type: 'low_health', data: { currentHealth: this.player.getHealth() }, timestamp: Date.now() });
        this.stopCombat();
        return;
      }
      
      switch (mode) {
        case 'aggressive':
          this.executeAggressiveCombat(target);
          break;
        case 'defensive':
          this.executeDefensiveCombat(target);
          break;
        case 'passive':
          // 不主动攻击，只防御
          break;
      }
    }, 100);
  }
  
  private stopCombatLoop() {
    if (this.combatLoop) {
      clearInterval(this.combatLoop);
      this.combatLoop = null;
    }
  }
  
  private executeAggressiveCombat(target: Entity) {
    const distance = this.calculateDistance(this.player.pos, target.pos);
    if (distance > 3) {
      this.moveTo(target.pos);
    }
    if (distance <= 3) {
      this.player.attack(target);
    }
  }
  
  private executeDefensiveCombat(target: Entity) {
    const distance = this.calculateDistance(this.player.pos, target.pos);
    const keepDistance = 4;
    
    if (distance < keepDistance - 1) {
      this.moveAway(target.pos);
    } else if (distance > keepDistance + 1) {
      this.moveTo(target.pos);
    }
    
    if (distance <= keepDistance) {
      this.player.attack(target);
    }
  }
  
  async interactEntity(entityId: string, action: 'feed' | 'breed' | 'trade' | 'tame' | 'shear' | 'milk', tradeIndex?: number): Promise<ResultEnvelope<InteractEntityData>> {
    const entity = getEntityById(entityId);
    if (!entity) {
      return {
        success: false,
        error: { code: 'NOT_FOUND', message: `实体不存在: ${entityId}` },
        meta: { duration: 20 }
      };
    }
    
    const distance = this.calculateDistance(this.player.pos, entity.pos);
    if (distance > 5) {
      return {
        success: false,
        error: { code: 'TOO_FAR', message: `距离太远: ${distance.toFixed(1)}格` },
        meta: { duration: 10 }
      };
    }
    
    switch (action) {
      case 'feed': {
        if (!this.isFeedable(entity)) {
          return {
            success: false,
            error: { code: 'NOT_FOUND', message: '该实体不可喂食' },
            meta: { duration: 30 }
          };
        }
        const food = this.findFoodForEntity(entity);
        if (!food) {
          return {
            success: false,
            error: { code: 'NO_FOOD', message: '背包中没有适合该实体的食物' },
            meta: { duration: 30 }
          };
        }
        this.player.feedEntity(entity, food);
        return {
          success: true,
          data: { entityName: entity.name, details: { foodUsed: food.name } },
          meta: { duration: 500, cost: { hunger: 0.1 } }
        };
      }
      
      case 'breed': {
        if (!this.isBreedable(entity)) {
          return {
            success: false,
            error: { code: 'INVALID_PARAMS', message: '该实体不可繁殖' },
            meta: { duration: 30 }
          };
        }
        const food = this.findFoodForEntity(entity);
        if (!food) {
          return {
            success: false,
            error: { code: 'NO_FOOD', message: '背包中没有适合该实体的食物' },
            meta: { duration: 30 }
          };
        }
        this.player.breedEntity(entity, food);
        return {
          success: true,
          data: { entityName: entity.name, details: { babySpawned: true } },
          meta: { duration: 800, cost: { hunger: 0.2 } }
        };
      }
      
      case 'trade': {
        if (entity.type !== 'villager') {
          return {
            success: false,
            error: { code: 'INVALID_PARAMS', message: '该实体不是村民' },
            meta: { duration: 20 }
          };
        }
        const trades = this.getVillagerTrades(entity);
        if (tradeIndex === undefined || tradeIndex >= trades.length) {
          return {
            success: false,
            error: { code: 'INVALID_PARAMS', message: `交易选项无效: ${tradeIndex}` },
            meta: { duration: 20 }
          };
        }
        const trade = trades[tradeIndex];
        this.player.tradeWithVillager(entity, tradeIndex);
        return {
          success: true,
          data: {
            entityName: entity.name,
            details: { tradeDetails: { bought: trade.sell, sold: trade.buy, times: 1 } }
          },
          meta: { duration: 1200, cost: {} }
        };
      }
      
      case 'tame': {
        if (!this.isTameable(entity)) {
          return {
            success: false,
            error: { code: 'INVALID_PARAMS', message: '该实体不可驯服' },
            meta: { duration: 30 }
          };
        }
        const result = this.player.tameEntity(entity);
        return {
          success: true,
          data: { entityName: entity.name, details: { tamed: result.tamed, attempts: result.attempts } },
          meta: { duration: 2000, cost: { hunger: 0.3 } }
        };
      }
      
      case 'shear': {
        if (!this.isShearable(entity)) {
          return {
            success: false,
            error: { code: 'INVALID_PARAMS', message: '该实体不可剪毛' },
            meta: { duration: 30 }
          };
        }
        const wool = this.player.shearEntity(entity);
        return {
          success: true,
          data: { entityName: entity.name, details: { woolDropped: wool } },
          meta: { duration: 600, cost: { durability: 1 } }
        };
      }
      
      case 'milk': {
        if (!this.isMilkable(entity)) {
          return {
            success: false,
            error: { code: 'INVALID_PARAMS', message: '该实体不可挤奶' },
            meta: { duration: 30 }
          };
        }
        this.player.milkEntity(entity);
        return {
          success: true,
          data: { entityName: entity.name, details: { milkObtained: true } },
          meta: { duration: 600, cost: {} }
        };
      }
    }
  }
  
  async leadEntity(entityId: string, action: 'lead' | 'release'): Promise<ResultEnvelope<LeadEntityData>> {
    const entity = getEntityById(entityId);
    if (!entity) {
      return {
        success: false,
        error: { code: 'NOT_FOUND', message: `实体不存在: ${entityId}` },
        meta: { duration: 20 }
      };
    }
    
    const distance = this.calculateDistance(this.player.pos, entity.pos);
    if (distance > 5) {
      return {
        success: false,
        error: { code: 'TOO_FAR', message: `距离太远: ${distance.toFixed(1)}格` },
        meta: { duration: 10 }
      };
    }
    
    if (action === 'lead') {
      if (!this.isLeashable(entity)) {
        return {
          success: false,
          error: { code: 'INVALID_PARAMS', message: '该实体不可牵引' },
          meta: { duration: 30 }
        };
      }
      this.player.leadEntity(entity);
      return {
        success: true,
        data: { entityName: entity.name },
        meta: { duration: 500, cost: { hunger: 0.1 } }
      };
    } else {
      this.player.releaseEntity(entity);
      return {
        success: true,
        data: { entityName: entity.name },
        meta: { duration: 300, cost: {} }
      };
    }
  }
  
  private calculateDistance(a: Position, b: Position): number {
    return Math.sqrt(Math.pow(a.x - b.x, 2) + Math.pow(a.y - b.y, 2) + Math.pow(a.z - b.z, 2));
  }
}
```

## 6. 总结

### 6.1 核心设计

| 组件 | 职责 |
|---|---|
| **LLM（大脑）** | 切换战斗模式，调用生物交互工具 |
| **战斗模式** | 持续状态，执行AI自动执行 |
| **状态变化通知** | 目标死亡/血量过低时通知LLM |
| **生物交互** | 通过 action 参数区分喂食/繁殖/交易/驯服/剪毛/挤奶 |

### 6.2 关键特性

1. **模式切换**：LLM切换战斗模式（aggressive/defensive/passive）
2. **持续执行**：执行AI自动执行战斗逻辑，不需要每次攻击返回
3. **状态变化通知**：目标死亡/血量过低时主动通知LLM
4. **精简接口**：4 个工具覆盖所有生物交互操作，降低 LLM 决策复杂度

### 6.3 工具列表

| 类别 | 工具 |
|---|---|
| 战斗模式 | set_combat_mode, stop_combat |
| 生物交互 | interact_entity |
| 实体操作 | lead_entity |
