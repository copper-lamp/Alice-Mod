# 感知系统设计 v2（Perception System Design v2）

> 版本：v2.0
> 日期：2026-07-01
> 架构：执行AI（身体）→ 基础状态注入 → LLM（大脑）→ 环境感知工具调用

---

## 0. 设计原则

### 0.1 核心理念

**上下文只注入基础状态，环境感知靠 LLM 主动调工具**

```
┌─────────────────────────────────────────────────────┐
│  执行AI（身体）                                       │
│  - 持续采集基础状态                                   │
│  - 基础状态自动注入 LLM 上下文                        │
└────────────────┬────────────────────────────────────┘
                 │
                 ▼
        ┌────────────────┐
        │  基础状态注入    │  ← 固定注入，每轮对话前
        │  - 生命/饥饿    │
        │  - 防御/装备    │
        │  - 位置/维度    │
        │  - 背包摘要     │
        │  - 特殊状况     │
        └────────┬───────┘
                 │
                 ▼
        ┌────────────────┐
        │  LLM（大脑）    │
        │  - 接收基础状态  │
        │  - 需要环境信息时│
        │    主动调工具    │
        └────────┬───────┘
                 │
                 ▼
        ┌────────────────┐
        │  环境感知工具    │  ← LLM 按需调用
        │  - 附近实体     │
        │  - 方块详情     │
        │  - 容器内容     │
        │  - 世界状态     │
        └────────────────┘
```

### 0.2 两类感知

| 类型 | 注入方式 | 频率 | 内容 |
|---|---|---|---|
| **基础状态** | 自动注入上下文 | 每轮对话前 | 生命/饥饿/防御/位置/背包/特殊状况 |
| **环境感知** | LLM 主动调工具 | 按需 | 附近实体/方块/容器/世界状态 |

## 1. 基础状态注入（自动）

### 1.1 数据结构

```typescript
interface BasicState {
  // 生命状态
  health: {
    current: number;      // 当前血量（0-20）
    max: number;          // 最大血量（通常 20）
    hunger: number;       // 饥饿值（0-20）
    saturation: number;   // 饱和度（0-20）
  };
  
  // 防御状态
  defense: {
    armor: number;        // 护甲值（0-20）
    toughness: number;    // 护甲韧性
    absorption: number;   // 伤害吸收
  };
  
  // 位置信息
  position: {
    x: number;
    y: number;
    z: number;
    dimension: 'overworld' | 'nether' | 'end';
    biome: string;        // 生物群系
    lightLevel: number;   // 光照等级（0-15）
  };
  
  // 装备
  equipment: {
    hand: Item | null;      // 主手
    offhand: Item | null;   // 副手
    helmet: Item | null;
    chestplate: Item | null;
    leggings: Item | null;
    boots: Item | null;
  };
  
  // 背包摘要
  inventory: {
    usedSlots: number;
    totalSlots: number;
    emptySlots: number;
    items: Array<{
      name: string;
      count: number;
    }>;
  };
  
  // 特殊状况
  status: {
    effects: string[];      // 药水效果名称
    isOnFire: boolean;
    isInWater: boolean;
    isInLava: boolean;
    isSneaking: boolean;
    isSleeping: boolean;
  };
}
```

### 1.2 序列化格式

**自然语言格式**（注入到 LLM 上下文）：

```
【你的状态】
生命: 18/20 饥饿: 15/20 饱和度: 3.2
防御: 护甲 12 韧性 2 吸收 0
位置: (128, 64, -256) 主世界 平原 光照 12
装备: 主手=铁剑 副手=盾牌 头盔=铁头盔 胸甲=铁胸甲 腿甲=铁腿甲 靴子=铁靴子
背包: 23/36 (空位 13)
  - 苹果 x12
  - 圆石 x45
  - 橡木原木 x8
  - 铁锭 x3
  - ... (共 15 种物品)
状态: 无特殊状况
```

**Token 消耗**：约 150-200 tokens

### 1.3 注入策略

```typescript
function injectBasicState(state: BasicState): string {
  const lines = [
    '【你的状态】',
    `生命: ${state.health.current}/${state.health.max} 饥饿: ${state.health.hunger}/20 饱和度: ${state.health.saturation.toFixed(1)}`,
    `防御: 护甲 ${state.defense.armor} 韧性 ${state.defense.toughness} 吸收 ${state.defense.absorption}`,
    `位置: (${state.position.x.toFixed(1)}, ${state.position.y.toFixed(1)}, ${state.position.z.toFixed(1)}) ${state.position.dimension} ${state.position.biome} 光照 ${state.position.lightLevel}`,
  ];
  
  // 装备
  const equipment = [];
  if (state.equipment.hand) equipment.push(`主手=${state.equipment.hand.name}`);
  if (state.equipment.offhand) equipment.push(`副手=${state.equipment.offhand.name}`);
  if (state.equipment.helmet) equipment.push(`头盔=${state.equipment.helmet.name}`);
  if (state.equipment.chestplate) equipment.push(`胸甲=${state.equipment.chestplate.name}`);
  if (state.equipment.leggings) equipment.push(`腿甲=${state.equipment.leggings.name}`);
  if (state.equipment.boots) equipment.push(`靴子=${state.equipment.boots.name}`);
  if (equipment.length > 0) {
    lines.push(`装备: ${equipment.join(' ')}`);
  } else {
    lines.push('装备: 无');
  }
  
  // 背包
  lines.push(`背包: ${state.inventory.usedSlots}/${state.inventory.totalSlots} (空位 ${state.inventory.emptySlots})`);
  state.inventory.items.slice(0, 10).forEach(item => {
    lines.push(`  - ${item.name} x${item.count}`);
  });
  if (state.inventory.items.length > 10) {
    lines.push(`  - ... (共 ${state.inventory.items.length} 种物品)`);
  }
  
  // 状态
  const statusList = [];
  if (state.status.effects.length > 0) statusList.push(...state.status.effects);
  if (state.status.isOnFire) statusList.push('着火');
  if (state.status.isInWater) statusList.push('在水中');
  if (state.status.isInLava) statusList.push('在岩浆中');
  if (state.status.isSneaking) statusList.push('潜行中');
  if (state.status.isSleeping) statusList.push('睡觉中');
  
  if (statusList.length > 0) {
    lines.push(`状态: ${statusList.join(', ')}`);
  } else {
    lines.push('状态: 无特殊状况');
  }
  
  return lines.join('\n');
}
```

```typescript
async function chatWithLLM(userMessage: string): Promise<string> {
  // 1. 采集基础状态
  const basicState = collectBasicState();
  
  // 2. 序列化
  const stateText = injectBasicState(basicState);
  
  // 3. 构造消息
  const messages = [
    { role: 'system', content: SYSTEM_PROMPT },
    { role: 'system', content: stateText },  // ← 每轮对话前注入
    { role: 'user', content: userMessage }
  ];
  
  // 4. 调用 LLM

  return response;
}
```

## 2. 环境感知工具（LLM 主动调用）

### 2.1 工具列表

| 工具名 | 功能 | 参数 | 返回 |
|---|---|---|---|
| `look_around` | 查看附近实体/方块 | `radius?: number` (默认 16), `filter?: {hostile?, passive?, player?, item?, block?}` | 实体和方块列表（类型、距离、血量、是否敌对） |
| `look_at_block` | 查看指定坐标方块 | `x, y, z, radius?: number` | 方块名、NBT、是否有方块实体 |
| `look_in_container` | 查看容器内容 | `x, y, z` | 容器内物品列表 |
| `look_time_weather` | 查看时间天气 | — | 世界时间、是否白天、天气 |
| `look_online_players` | 查看在线玩家 | — | 玩家列表（名字、位置） |

## 3. 工具实现

### 3.1 工具注册

```typescript
// 注册工具
function registerPerceptionTools() {
  // look_around
  registerTool('look_around', {
    description: '查看附近实体和方块（生物、玩家、掉落物、方块）',
    parameters: {
      radius: { type: 'number', default: 16, max: 64 },
      filter: { type: 'object', optional: true }
    },
    execute: async (params) => {
      const radius = params.radius || 16;
      const filter = params.filter || {};
      const playerPos = agent.player.pos;
      
      // 查询实体
      let entities = [];
      if (!filter.block) {
        const allEntities = mc.getEntities();
        entities = allEntities
          .map(entity => ({
            id: entity.uniqueId,
            type: entity.type,
            name: entity.name,
            distance: calculateDistance(playerPos, entity.pos),
            position: { x: entity.pos.x, y: entity.pos.y, z: entity.pos.z },
            health: entity.getHealth ? { current: entity.getHealth(), max: entity.getMaxHealth() } : undefined,
            isHostile: isHostile(entity.type),
            isPlayer: entity.isPlayer(),
            isItem: entity.isItemEntity(),
            itemStack: entity.isItemEntity() ? getItemStack(entity) : undefined
          }))
          .filter(e => {
            if (e.distance > radius || e.distance < 0.5) return false;
            if (filter.hostile && !e.isHostile) return false;
            if (filter.passive && e.isHostile) return false;
            if (filter.player && !e.isPlayer) return false;
            if (filter.item && !e.isItem) return false;
            return true;
          })
          .sort((a, b) => a.distance - b.distance)
          .slice(0, 50);  // 最多 50 个
      }
      
      // 查询方块
      let blocks = undefined;
      if (filter.block || (!filter.hostile && !filter.passive && !filter.player && !filter.item)) {
        const nearbyBlocks = scanNearbyBlocks(playerPos, radius);
        if (nearbyBlocks.length > 0) {
          blocks = nearbyBlocks;
        }
      }
      
      return { entities, blocks, total: entities.length + (blocks ? blocks.reduce((s, b) => s + b.count, 0) : 0) };
    }
  });
  
  // look_at_block
  registerTool('look_at_block', {
    description: '查看指定坐标的方块（支持批量查询附近方块）',
    parameters: {
      x: { type: 'number', required: true },
      y: { type: 'number', required: true },
      z: { type: 'number', required: true },
      radius: { type: 'number', optional: true, default: 0, max: 16 }
    },
    execute: async (params) => {
      const { x, y, z, radius = 0 } = params;
      
      // 批量查询
      if (radius > 0) {
        const blocks = scanNearbyBlocks({ x, y, z }, radius);
        return { blocks, total: blocks.reduce((s, b) => s + b.count, 0) };
      }
      
      // 单个方块查询
      const block = mc.getBlock(x, y, z);
      if (!block) return { error: '方块不存在' };
      
      return {
        name: block.name,
        displayName: block.name,
        position: { x: block.pos.x, y: block.pos.y, z: block.pos.z },
        isSolid: block.isSolid(),
        isLiquid: block.isLiquid(),
        isAir: block.isAir(),
        hasBlockEntity: block.hasBlockEntity(),
        nbt: block.hasBlockEntity() ? block.getNbt() : undefined
      };
    }
  });
  
  // look_in_container
  registerTool('look_in_container', {
    description: '查看容器（箱子、熔炉等）的内容',
    parameters: {
      x: { type: 'number', required: true },
      y: { type: 'number', required: true },
      z: { type: 'number', required: true }
    },
    execute: async (params) => {
      const block = mc.getBlock(params.x, params.y, params.z);
      if (!block || !block.hasBlockEntity()) {
        return { error: '不是容器' };
      }
      
      const blockEntity = block.getBlockEntity();
      const container = blockEntity.getContainer();
      if (!container) return { error: '无法获取容器' };
      
      const items = [];
      const allItems = container.getAllItems();
      allItems.forEach((item, slot) => {
        if (!item.isNull()) {
          items.push({
            slot,
            name: item.type,
            count: item.count,
            nbt: item.getNbt()
          });
        }
      });
      
      return {
        containerType: block.name,
        position: { x: params.x, y: params.y, z: params.z },
        totalSlots: container.getSize(),
        usedSlots: items.length,
        items
      };
    }
  });
  
  // look_time_weather
  registerTool('look_time_weather', {
    description: '查看世界时间和天气',
    parameters: {},
    execute: async () => {
      const world = mc.getWorld();
      return {
        worldTime: world.getTime(),
        dayTime: world.getTime() % 24000,
        isDay: world.isDay(),
        weather: world.getWeather(),
        difficulty: world.getDifficulty()
      };
    }
  });
  
  // look_online_players
  registerTool('look_online_players', {
    description: '查看在线玩家列表',
    parameters: {},
    execute: async () => {
      const players = mc.getPlayers().map(p => ({
        name: p.name,
        uuid: p.uuid,
        position: { x: p.pos.x, y: p.pos.y, z: p.pos.z },
        dimension: p.dimension,
        distance: calculateDistance(agent.player.pos, p.pos)
      }));
      return { players, total: players.length };
    }
  });
}
```

### 3.2 工具调用流程

```typescript
// LLM 调用工具
async function callTool(toolName: string, params: any): Promise<string> {
  const tool = getTool(toolName);
  if (!tool) return `错误: 工具 ${toolName} 不存在`;
  
  try {
    const result = await tool.execute(params);
    return formatToolResult(toolName, result);
  } catch (error) {
    return `错误: ${error.message}`;
  }
}

// 格式化工具结果
function formatToolResult(toolName: string, result: any): string {
  switch (toolName) {
    case 'look_around':
      return formatLookAroundResult(result);
    case 'look_at_block':
      return formatLookAtBlockResult(result);
    case 'look_in_container':
      return formatLookInContainerResult(result);
    // ...
    default:
      return JSON.stringify(result, null, 2);
  }
}

function formatLookAroundResult(result: LookAroundResult): string {
  const lines = [];
  const entityCount = result.entities ? result.entities.length : 0;
  const blockCount = result.blocks ? result.blocks.reduce((s, b) => s + b.count, 0) : 0;
  
  if (entityCount === 0 && blockCount === 0) {
    return '附近没有实体或方块';
  }
  
  lines.push(`附近环境 (${entityCount}个实体, ${blockCount}个方块):`);
  
  // 实体
  result.entities.forEach(entity => {
    let desc = `  - ${entity.name}`;
    if (entity.isHostile) desc += ' (敌对)';
    if (entity.isPlayer) desc += ' (玩家)';
    if (entity.isItem && entity.itemStack) {
      desc += ` ${entity.itemStack.name}x${entity.itemStack.count}`;
    }
    desc += ` ${entity.distance.toFixed(1)}格`;
    if (entity.health) {
      desc += ` 血量 ${entity.health.current}/${entity.health.max}`;
    }
    lines.push(desc);
  });
  
  // 方块
  if (result.blocks && result.blocks.length > 0) {
    const blockDesc = result.blocks.map(b => `${b.displayName} x${b.count}`).join(', ');
    lines.push(`  - 方块: ${blockDesc}`);
  }
  
  return lines.join('\n');
}

function formatLookAtBlockResult(result: any): string {
  if (result.blocks) {
    // 批量查询结果
    const lines = [`附近方块 (共${result.total}个):`];
    result.blocks.forEach(b => {
      lines.push(`  - ${b.displayName} x${b.count}`);
    });
    return lines.join('\n');
  }
  
  // 单个方块结果
  return `方块 (${result.position.x},${result.position.y},${result.position.z}): ${result.displayName} (${result.isSolid ? '固体' : '非固体'}, ${result.hasBlockEntity ? '有方块实体' : '无方块实体'})`;
}
```

## 5. 总结

### 5.1 核心设计

| 组件 | 职责 |
|---|---|
| **基础状态注入** | 自动注入生命/饥饿/防御/位置/背包/状态（~150 tokens） |
| **环境感知工具** | LLM 按需调用（look_around, look_at_block, look_in_container 等） |
| **工具注册系统** | 注册工具、执行工具、格式化结果 |

### 5.2 关键特性

1. **基础状态自动注入**：每轮对话前自动注入，LLM 始终知道"我是谁、我在哪、我有什么"
2. **环境感知工具化**：LLM 需要环境信息时主动调工具，避免上下文爆炸
3. **工具返回自然语言**：工具结果格式化成 LLM 易读的格式
4. **Token 可控**：基础状态 ~150 tokens，工具结果按需返回
