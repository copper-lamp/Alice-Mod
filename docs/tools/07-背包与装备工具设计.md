# 背包与装备工具设计（Inventory & Equipment Tools Design）

> 版本：v1.0
> 日期：2026-07-14
> 架构：LLM 调用工具 → 执行AI 执行操作 → 返回结果
> 协议：遵循 `00-工具协议规范.md` 统一接口契约

---

## 0. 设计原则

### 0.1 核心概念

**背包与装备操作由执行AI负责，LLM 只调用工具**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 决定要整理/丢弃/取放/装备                          │
│  - 调用背包工具                                       │
└────────────────┬────────────────────────────────────┘
                 │ 调用工具
                 ▼
        ┌────────────────┐
        │  背包工具        │
        │  - drop_item    │
        │  - take/put     │
        │  - equip_item   │
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 背包整理算法  │
        │  - 物品移动      │
        │  - 装备切换      │
        └────────┬───────┘
                 │ 返回结果
                 ▼
        ┌────────────────┐
        │  工具返回        │
        │  - 成功/失败    │
        │  - 操作详情     │
        └────────────────┘
```

### 0.2 设计原则

| 原则 | 说明 |
|---|---|
| **一键操作** | 整理、丢弃等操作尽量一键完成 |
| **有反馈** | 工具返回操作结果（成功/失败/详情） |
| **可失败** | 背包满、物品不存在、距离太远等 |
| **简单直接** | 工具粒度适中，避免过度设计 |

## 1. 工具列表

### 1.1 丢弃

| 工具名 | 功能 | 参数 | 返回 |
|---|---|---|---|
| `drop_item` | 丢弃物品/向指定实体丢弃物品 | `item_name?: string`, `count?: number`（不指定 count 丢全部）, `target_entity?: string` | `ResultEnvelope<DropItemData>` |

### 1.2 容器操作

| 工具名 | 功能 | 参数 | 返回 |
|---|---|---|---|
| `take_from_container` | 从容器取物 | `x: number, y: number, z: number`, `item_name?: string`, `count?: number` | `ResultEnvelope<TakeFromContainerData>` |
| `put_to_container` | 向容器放物 | `x: number, y: number, z: number`, `item_name?: string`, `count?: number` | `ResultEnvelope<PutToContainerData>` |

### 1.3 装备操作

| 工具名 | 功能 | 参数 | 返回 |
|---|---|---|---|
| `equip_item` | 装备/卸下物品 | `item_name: string`, `slot: string`, `action: 'equip' \| 'unequip'` | `ResultEnvelope<EquipItemData>` |

## 2. 工具定义

### 2.1 drop_item（丢弃物品）

#### 参数规范（ParameterSchema）

| 属性 | 类型 | 必填 | 描述 | 默认值 |
|------|------|------|------|--------|
| `item_name` | `string` | 否 | 物品名称（如 "cobblestone", "dirt"），不填则丢弃全部物品 | - |
| `count` | `number` | 否 | 丢弃数量，不填或超出则丢弃全部 | - |
| `target_entity` | `string` | 否 | 目标实体ID，向指定实体丢弃物品（可选） | - |

#### 返回定义（ResultEnvelope<DropItemData>）

```typescript
interface DropItemData {
  /** 实际丢弃数量 */
  droppedCount?: number;
}
```

**错误码**：

| 错误码 | 条件 |
|--------|------|
| `ITEM_NOT_FOUND` | 背包中找不到指定物品 |
| `INTERNAL_ERROR` | 内部错误（如数量不足） |

**示例调用**：

```
LLM 调用: drop_item(item_name="cobblestone", count=32)
返回:
{
  "success": true,
  "data": {
    "droppedCount": 32
  },
  "meta": {
    "duration": 150,
    "cost": { "hunger": 0.5 }
  }
}
```

**自然语言返回**：

```
丢弃成功: 圆石 x32
```

**向实体丢弃**：

```
LLM 调用: drop_item(item_name="diamond", count=5, target_entity="villager_123")
返回:
{
  "success": true,
  "data": {
    "droppedCount": 5
  },
  "meta": {
    "duration": 200
  }
}
```

**失败示例**：

```
{
  "success": false,
  "error": {
    "code": "ITEM_NOT_FOUND",
    "message": "背包中找不到物品 diamond"
  },
  "meta": {
    "duration": 30
  }
}
```

### 3.2 执行AI接口

```typescript
// 执行AI接口（背包操作）
interface ExecutionAI {
  // 丢弃
  dropItem(itemName?: string, count?: number, targetEntity?: string): Promise<ResultEnvelope<DropItemData>>;
  
  // 容器操作
  takeFromContainer(x: number, y: number, z: number, itemName?: string, count?: number): Promise<ResultEnvelope<TakeFromContainerData>>;
  putToContainer(x: number, y: number, z: number, itemName?: string, count?: number): Promise<ResultEnvelope<PutToContainerData>>;
  
  // 装备操作
  equipItem(itemName: string, slot: 'helmet' | 'chestplate' | 'leggings' | 'boots' | 'elytra', action: 'equip' | 'unequip'): Promise<ResultEnvelope<EquipItemData>>;
}
```

### 3.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  
  constructor(player: Player) {
    this.player = player;
  }
  
  async dropItem(itemName?: string, count?: number, targetEntity?: string): Promise<ResultEnvelope<DropItemData>> {
    const inventory = this.player.getInventory();
    const startTime = Date.now();
    
    if (!itemName) {
      // 不指定物品名，丢弃全部
      const allItems = inventory.getAllItems();
      let totalDropped = 0;
      
      for (const item of allItems) {
        if (!item.isNull()) {
          const slot = inventory.findItem(item.name);
          this.player.dropItem(slot, item.count, targetEntity);
          totalDropped += item.count;
        }
      }
      
      return {
        success: true,
        data: { droppedCount: totalDropped },
        meta: { duration: Date.now() - startTime, cost: { hunger: 0.5 } }
      };
    }
    
    const slot = inventory.findItem(itemName);
    if (slot === -1) {
      return {
        success: false,
        error: { code: 'ITEM_NOT_FOUND', message: `背包中找不到物品 ${itemName}` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const item = inventory.getItem(slot);
    const dropCount = count || item.count;
    
    if (dropCount > item.count) {
      return {
        success: false,
        error: { code: 'INTERNAL_ERROR', message: `物品不足：需要 ${dropCount}，实际 ${item.count}` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    this.player.dropItem(slot, dropCount, targetEntity);
    
    return {
      success: true,
      data: { droppedCount: dropCount },
      meta: { duration: Date.now() - startTime, cost: { hunger: 0.5 } }
    };
  }
  
  async takeFromContainer(x: number, y: number, z: number, itemName?: string, count?: number): Promise<ResultEnvelope<TakeFromContainerData>> {
    const startTime = Date.now();
    const distance = this.calculateDistance(this.player.pos, {x, y, z});
    if (distance > 5) {
      return {
        success: false,
        error: { code: 'TOO_FAR', message: '距离容器太远，无法操作' },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const block = mc.getBlock(x, y, z);
    if (!block || !block.hasBlockEntity()) {
      return {
        success: false,
        error: { code: 'CONTAINER_NOT_FOUND', message: `坐标 (${x}, ${y}, ${z}) 处不存在容器` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const container = block.getBlockEntity().getContainer();
    const inventory = this.player.getInventory();
    
    if (!itemName) {
      // 不指定物品名，取全部
      let totalTaken = 0;
      const allItems = container.getAllItems();
      for (const item of allItems) {
        if (!item.isNull()) {
          const slot = container.findItem(item.name);
          const takeCount = count || item.count;
          container.transferItem(slot, inventory, takeCount);
          totalTaken += takeCount;
        }
      }
      return {
        success: true,
        data: { takenCount: totalTaken },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const slot = container.findItem(itemName);
    if (slot === -1) {
      return {
        success: false,
        error: { code: 'ITEM_NOT_FOUND', message: `容器中找不到物品 ${itemName}` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const item = container.getItem(slot);
    const takeCount = count || item.count;
    
    if (!inventory.hasRoomFor(itemName, takeCount)) {
      return {
        success: false,
        error: { code: 'INVENTORY_FULL', message: '背包已满，无法放入更多物品' },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    container.transferItem(slot, inventory, takeCount);
    
    return {
      success: true,
      data: { takenCount: takeCount },
      meta: { duration: Date.now() - startTime }
    };
  }
  
  async putToContainer(x: number, y: number, z: number, itemName?: string, count?: number): Promise<ResultEnvelope<PutToContainerData>> {
    const startTime = Date.now();
    const distance = this.calculateDistance(this.player.pos, {x, y, z});
    if (distance > 5) {
      return {
        success: false,
        error: { code: 'TOO_FAR', message: '距离容器太远，无法操作' },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const block = mc.getBlock(x, y, z);
    if (!block || !block.hasBlockEntity()) {
      return {
        success: false,
        error: { code: 'CONTAINER_NOT_FOUND', message: `坐标 (${x}, ${y}, ${z}) 处不存在容器` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const container = block.getBlockEntity().getContainer();
    const inventory = this.player.getInventory();
    
    if (!itemName) {
      // 不指定物品名，放全部
      let totalPut = 0;
      const allItems = inventory.getAllItems();
      for (const item of allItems) {
        if (!item.isNull()) {
          const slot = inventory.findItem(item.name);
          const putCount = count || item.count;
          inventory.transferItem(slot, container, putCount);
          totalPut += putCount;
        }
      }
      return {
        success: true,
        data: { putCount: totalPut },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const slot = inventory.findItem(itemName);
    if (slot === -1) {
      return {
        success: false,
        error: { code: 'ITEM_NOT_FOUND', message: `背包中找不到物品 ${itemName}` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    const item = inventory.getItem(slot);
    const putCount = count || item.count;
    
    if (!container.hasRoomFor(itemName, putCount)) {
      return {
        success: false,
        error: { code: 'CONTAINER_FULL', message: '容器已满，无法放入更多物品' },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    inventory.transferItem(slot, container, putCount);
    
    return {
      success: true,
      data: { putCount: putCount },
      meta: { duration: Date.now() - startTime }
    };
  }
  
  async equipItem(itemName: string, slot: 'helmet' | 'chestplate' | 'leggings' | 'boots' | 'elytra', action: 'equip' | 'unequip'): Promise<ResultEnvelope<EquipItemData>> {
    const startTime = Date.now();
    
    if (action === 'equip') {
      const inventory = this.player.getInventory();
      const itemSlot = inventory.findItem(itemName);
      
      if (itemSlot === -1) {
        return {
          success: false,
          error: { code: 'ITEM_NOT_FOUND', message: `背包中找不到物品 ${itemName}` },
          meta: { duration: Date.now() - startTime }
        };
      }
      
      const mappedSlot = slot === 'elytra' ? 'chestplate' : slot;
      this.player.equipArmor(mappedSlot, itemSlot);
      
      return {
        success: true,
        data: { equippedItem: itemName },
        meta: { duration: Date.now() - startTime }
      };
      
    } else {
      const mappedSlot = slot === 'elytra' ? 'chestplate' : slot;
      const equipped = this.player.getEquippedArmor(mappedSlot);
      
      if (!equipped) {
        return {
          success: false,
          error: { code: 'NOT_FOUND', message: '当前槽位未装备物品' },
          meta: { duration: Date.now() - startTime }
        };
      }
      
      const inventory = this.player.getInventory();
      if (!inventory.hasRoomFor(equipped.name, 1)) {
        return {
          success: false,
          error: { code: 'INVENTORY_FULL', message: '背包已满，无法放入卸下的装备' },
          meta: { duration: Date.now() - startTime }
        };
      }
      
      this.player.unequipArmor(mappedSlot);
      
      return {
        success: true,
        data: { unequippedItem: equipped.name },
        meta: { duration: Date.now() - startTime }
      };
    }
  }
  
  private calculateDistance(a: Position, b: Position): number {
    return Math.sqrt(
      Math.pow(a.x - b.x, 2) +
      Math.pow(a.y - b.y, 2) +
      Math.pow(a.z - b.z, 2)
    );
  }
}
```

## 5. 总结

### 5.1 核心设计

| 组件 | 职责 |
|---|---|
| **LLM（大脑）** | 决定要丢弃/取放/装备，调用背包工具 |
| **背包工具** | 封装背包操作，返回执行结果 |
| **执行AI（假人）** | 负责具体背包操作（丢弃、移动、装备等） |

### 5.2 关键特性

1. **精简接口**：4 个工具覆盖所有背包操作，降低 LLM 决策复杂度
2. **有反馈**：工具返回操作结果（成功/失败/详情）
3. **可失败**：背包满、物品不存在、距离太远等
4. **灵活丢弃**：`drop_item` 支持指定物品/全部物品/向实体丢弃

### 5.3 工具列表

| 类别 | 工具 |
|---|---|
| 丢弃 | drop_item |
| 容器操作 | take_from_container, put_to_container |
| 装备操作 | equip_item |
