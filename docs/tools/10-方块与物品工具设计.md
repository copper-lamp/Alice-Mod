# 方块与物品工具设计（Block & Item Tools Design）

> 版本：v1.0
> 日期：2026-07-01
> 架构：LLM 调用工具 → 执行AI 执行操作 → 返回结果
> 协议版本：遵循 `00-工具协议规范.md` v1.0

---

## 0. 设计原则

### 0.1 核心概念

**方块操作由执行AI负责，LLM 只调用工具**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 决定要挖/放/用方块                                 │
│  - 调用方块工具                                       │
└────────────────┬────────────────────────────────────┘
                 │ 调用工具
                 ▼
        ┌────────────────┐
        │  方块工具        │
        │  - mine_block   │
        │  - place_block  │
        │  - use_block    │
        │  - area_operation │
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 自动选择工具  │
        │  - 自动寻路      │
        │  - 批量操作      │
        └────────┬───────┘
                 │ 返回结果
                 ▼
        ┌────────────────┐
        │  工具返回        │
        │  - 成功/失败    │
        │  - 操作详情     │
        └────────────────┘
```

### 0.2 设计原则

| 原则 | 说明 |
|---|---|
| **自动选择** | 执行AI自动选择合适工具（挖石头用镐、挖土用锹） |
| **批量操作** | 支持区域填充、批量破坏 |
| **有反馈** | 工具返回操作结果（成功/失败/详情） |
| **可失败** | 工具损坏、方块不可破坏、距离太远等 |

## 1. 工具列表

| 工具名 | 功能 | 参数 | 返回 |
|---|---|---|---|
| `mine_block` | 挖掘方块 | `x, y, z, options?` | `ResultEnvelope<MineBlockData>` |
| `place_block` | 放置方块 | `x, y, z, block_name, facing?` | `ResultEnvelope<PlaceBlockData>` |

## 2. 工具定义

### 2.1 mine_block（挖掘方块）

**ParameterSchema**（遵循 `00-工具协议规范.md` §2.2）：

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|------|------|--------|
| `x` | `number` | 是 | 方块的 X 坐标 | - |
| `y` | `number` | 是 | 方块的 Y 坐标 | - |
| `z` | `number` | 是 | 方块的 Z 坐标 | - |
| `options.tool` | `string` | 否 | 指定使用的工具名称（如 `iron_pickaxe`），不填则自动选择 | 自动选择 |
| `options.silkTouch` | `boolean` | 否 | 是否启用精准采集 | `false` |

**ResultEnvelope**：

```typescript
// data: MineBlockData
interface MineBlockData {
  blockName: string;                          // 挖掘的方块名称
  drops: Array<{ name: string; count: number }>;  // 掉落物列表
  toolUsed: string;                           // 使用的工具
  toolDurability: number;                     // 工具剩余耐久
}
```

**错误码**：

| 错误码 | 含义 | 说明 |
|--------|------|------|
| `BLOCK_NOT_FOUND` | 方块不存在 | 指定位置没有方块 |
| `NOT_BREAKABLE` | 方块不可破坏 | 如基岩、命令方块 |
| `WEAPON_BROKEN` | 工具损坏 | 使用过程中工具耐久耗尽 |
| `TOO_FAR` | 距离太远 | 超出交互距离（默认 5 格） |
| `INTERNAL_ERROR` | 内部错误 | 未预期的运行时错误 |

**示例调用**：

```
LLM 调用: mine_block(x=128, y=64, z=-256)
返回:
{
  "success": true,
  "data": {
    "blockName": "stone",
    "drops": [{ "name": "cobblestone", "count": 1 }],
    "toolUsed": "iron_pickaxe",
    "toolDurability": 245
  },
  "meta": {
    "duration": 1500,
    "cost": { "hunger": 0.5 }
  }
}
```

**自然语言返回**：

```
挖掘成功: 石头
掉落: 圆石 x1
工具: 铁镐 (耐久 245)
```

### 2.2 place_block（放置方块）

**ParameterSchema**：

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|------|------|--------|
| `x` | `number` | 是 | 放置位置的 X 坐标 | - |
| `y` | `number` | 是 | 放置位置的 Y 坐标 | - |
| `z` | `number` | 是 | 放置位置的 Z 坐标 | - |
| `block_name` | `string` | 是 | 要放置的方块名称（如 `stone`, `torch`） | - |
| `facing` | `string` | 否 | 朝向，枚举值：`north`, `south`, `east`, `west`, `up`, `down` | - |

**ResultEnvelope**：

**错误码**：

| 错误码 | 含义 | 说明 |

| `CANNOT_PLACE` | 无法放置方块 | 放置条件不满足（如无附着面） |
| `TOO_FAR` | 距离太远 | 超出交互距离 |
| `INTERNAL_ERROR` | 内部错误 | 未预期的运行时错误 |

```
  "data": {
    "blockName": "torch",
    "position": { "x": 128, "y": 64, "z": -256 }
  },
  "meta": {
    "duration": 800,
    "cost": { "hunger": 0.2 }

```

**自然语言返回**：

```

```

### 2.4 area_operation（区域操作）

**ParameterSchema**：

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|------|------|--------|
| `mode` | `string` | 是 | 操作模式，枚举值：`fill`, `clear`, `break`, `vein` | - |
| `from.x` | `number` | 是 | 区域起点的 X 坐标 | - |
| `from.y` | `number` | 是 | 区域起点的 Y 坐标 | - |
| `from.z` | `number` | 是 | 区域起点的 Z 坐标 | - |
| `to.x` | `number` | 是 | 区域终点的 X 坐标 | - |
| `to.y` | `number` | 是 | 区域终点的 Y 坐标 | - |
| `to.z` | `number` | 是 | 区域终点的 Z 坐标 | - |
| `block_name` | `string` | 否 | 方块名称（`fill` 模式需要） | - |
| `radius` | `number` | 否 | 搜索半径（`vein` 模式使用） | `5` |

**ResultEnvelope**：

```typescript
// data: AreaOperationData
interface AreaOperationData {
  blocksAffected: number;                     // 影响的方块数量
  drops?: Array<{ name: string; count: number }>;  // 掉落物列表（break/vein 模式）
  area?: { width: number; height: number; depth: number };  // 操作区域大小
  oreName?: string;                           // 矿脉名称（vein 模式）
}
```

**错误码**：

| 错误码 | 含义 | 说明 |
|--------|------|------|
| `ITEM_NOT_FOUND` | 背包中无此方块 | 填充模式指定方块不在背包中 |
| `TOO_LARGE` | 区域体积过大 | 超过最大限制（默认 10000 方块） |
| `NO_ORE` | 未找到矿脉 | 连锁挖掘模式起始点不是矿石 |
| `WEAPON_BROKEN` | 工具损坏 | 批量操作过程中工具损坏 |
| `INTERNAL_ERROR` | 内部错误 | 未预期的运行时错误 |

**示例调用（填充）**：

```

  "meta": {
    "duration": 5000,
    "cost": { "hunger": 2.0 }

```

### 2.5 collect_block（自动采集方块）

**ParameterSchema：**

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|:----:|------|:------:|
| `block_name` | `string` | 是 | 要采集的方块名称，支持模糊匹配 | - |
| `count` | `number` | 是 | 需要采集的数量 | - |
| `rules.modify_environment` | `boolean` | 否 | 是否允许修改环境（如垫脚石、挖开障碍物） | `false` |
| `budget_time_ms` | `number` | 否 | 预算时间（毫秒），超时未采够或方块耗尽时返回警告 | 不限时 |
| `bot_name` | `string` | 否 | 假人名称（多假人时必须指定） | - |

**ResultEnvelope：**

```typescript
// data: CollectBlockData
interface CollectBlockData {
  blockName: string;           // 采集的方块名称
  collected: number;           // 实际采集数量
  target: number;              // 目标采集数量
  warning?: string;            // 警告信息（预算时间耗尽/方块耗尽时返回）
  positions: Array<{           // 已采集的方块位置列表
    x: number;
    y: number;
    z: number;
  }>;
}
```

**执行逻辑：**

1. 使用螺旋扫描在附近查找目标方块（半径 64 格，最多 10 个位置）
2. 移动到每个方块位置，使用 BlockOperationEngine 挖掘
3. 循环查找→挖掘，直到达到目标数量
4. 每轮查找后检查以下条件并返回警告：
   - **预算时间耗尽**：已耗时超过 `budget_time_ms` 但未采够 → 返回 `warning`
   - **方块耗尽**：附近已找不到目标方块 → 返回 `warning`
5. 环境修改规则（`modify_environment`）传递给执行AI，影响寻路和挖掘行为

**错误码：**

| 错误码 | 含义 | 说明 |
|--------|------|------|
| `NOT_FOUND` | 假人未找到 | 未指定或不在线 |
| `INTERNAL_ERROR` | 内部错误 | 未预期的运行时错误 |

**示例调用（成功）**：

```
LLM 调用: collect_block(block_name="stone", count=32)
返回:
{
  "success": true,
  "data": {
    "blockName": "stone",
    "collected": 32,
    "target": 32,
    "positions": [
      { "x": 130, "y": 64, "z": -258 },
      { "x": 131, "y": 64, "z": -258 }
    ]
  },
  "meta": { "duration": 45000 }
}
```

**自然语言返回**：

```
采集完成: 石头 32/32 个
```

**示例调用（预算时间耗尽）**：

```
LLM 调用: collect_block(block_name="diamond_ore", count=5, budget_time_ms=60000)
返回:
{
  "success": true,
  "data": {
    "blockName": "diamond_ore",
    "collected": 2,
    "target": 5,
    "warning": "预算时间耗尽，仅采集到 2/5 个 diamond_ore",
    "positions": [
      { "x": 128, "y": 12, "z": -256 }
    ]
  },
  "meta": { "duration": 60000 }
}
```

**自然语言返回**：

```
警告: 预算时间耗尽，仅采集到 2/5 个钻石矿
```

**示例调用（方块耗尽）**：

```
LLM 调用: collect_block(block_name="ancient_debris", count=3)
返回:
{
  "success": true,
  "data": {
    "blockName": "ancient_debris",
    "collected": 1,
    "target": 3,
    "warning": "附近已无 ancient_debris，共采集到 1 个（目标 3 个）",
    "positions": [
      { "x": 128, "y": 15, "z": -256 }
    ]
  },
  "meta": { "duration": 30000 }
}
```

**自然语言返回**：

```
警告: 附近已无远古残骸，共采集到 1 个（目标 3 个）
```

## 3. 工具实现

### 3.1 工具注册

遵循 `00-工具协议规范.md` §5 的 `ToolRegistry.register` 格式：

```typescript
// 注册方块工具
function registerBlockTools() {
  // mine_block
  ToolRegistry.register({
    name: 'mine_block',
    description: '挖掘指定坐标的方块，支持自动选择工具和精准采集',
    parameterSchema: {
      type: 'object',
      properties: {
        x: { type: 'number', description: '方块的 X 坐标' },
        y: { type: 'number', description: '方块的 Y 坐标' },
        z: { type: 'number', description: '方块的 Z 坐标' },

          }
        }
      },
      required: ['x', 'y', 'z']
    },
    async invoke(params) {
      return await executionAI.mineBlock(params.x, params.y, params.z, params.options);
    }
  });
  
  // place_block
  ToolRegistry.register({
    name: 'place_block',
    description: '在指定位置放置方块，可指定朝向',
    parameterSchema: {
      type: 'object',
      properties: {
        x: { type: 'number', description: '放置位置的 X 坐标' },
        y: { type: 'number', description: '放置位置的 Y 坐标' },
        z: { type: 'number', description: '放置位置的 Z 坐标' },
        block_name: { type: 'string', description: '要放置的方块名称' },
        facing: {
          type: 'string',
          description: '朝向',

      return await executionAI.placeBlock(params.x, params.y, params.z, params.block_name, params.facing);
    }
  });
  
  // use_block
  ToolRegistry.register({
    name: 'use_block',
    description: '右键使用指定坐标的方块（如打开箱子、激活拉杆）',
    parameterSchema: {
      type: 'object',
      properties: {
        x: { type: 'number', description: '方块的 X 坐标' },
        y: { type: 'number', description: '方块的 Y 坐标' },
        z: { type: 'number', description: '方块的 Z 坐标' }
      },
      required: ['x', 'y', 'z']
    },
    async invoke(params) {
      return await executionAI.useBlock(params.x, params.y, params.z);
    }
  });
  
  // area_operation
  ToolRegistry.register({
    name: 'area_operation',
    description: '区域操作：填充（fill）、清空（clear）、破坏（break）、连锁挖掘（vein）',
    parameterSchema: {
      type: 'object',
      properties: {
        mode: {
          type: 'string',
          description: '操作模式',
          enum: ['fill', 'clear', 'break', 'vein']
        },
        from: {
          type: 'object',
          description: '区域起点',
          properties: {
            x: { type: 'number', description: 'X 坐标' },
            y: { type: 'number', description: 'Y 坐标' },
            z: { type: 'number', description: 'Z 坐标' }
          }
        },
        to: {
          type: 'object',
          description: '区域终点',
          properties: {
            x: { type: 'number', description: 'X 坐标' },
            y: { type: 'number', description: 'Y 坐标' },
            z: { type: 'number', description: 'Z 坐标' }
          }
        },
        block_name: { type: 'string', description: '方块名称（fill 模式需要）' },
        radius: { type: 'number', description: '搜索半径（vein 模式使用）', default: 5 }

      required: ['mode', 'from', 'to']
    },
    async invoke(params) {
      return await executionAI.areaOperation(params.mode, params.from, params.to, params.block_name, params.radius);
    }
  });
}
```

### 3.2 执行AI接口

```typescript
// 执行AI接口（方块操作）
interface ExecutionAI {
  // 单个操作
  mineBlock(x: number, y: number, z: number, options?: MineOptions): Promise<MineBlockResult>;
  placeBlock(x: number, y: number, z: number, blockName: string, facing?: string): Promise<PlaceBlockResult>;
  useBlock(x: number, y: number, z: number): Promise<UseBlockResult>;
  
  // 区域操作
  areaOperation(mode: 'fill' | 'clear' | 'break' | 'vein', from: {x,y,z}, to: {x,y,z}, blockName?: string, radius?: number): Promise<AreaOperationResult>;
}

// 挖掘选项
interface MineOptions {
  tool?: string;
  silk_touch?: boolean;
}
```

### 3.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  
  constructor(player: Player) {
    this.player = player;
  }
  
  async mineBlock(x: number, y: number, z: number, options?: MineOptions): Promise<MineBlockResult> {
    // 检查距离
    const distance = this.calculateDistance(this.player.pos, {x, y, z});
    if (distance > 5) {
      return { success: false, reason: 'too_far' };
    }
    
    // 获取方块
    const block = mc.getBlock(x, y, z);
    if (!block) {
      return { success: false, reason: 'block_not_found' };
    }
    
    // 检查是否可破坏
    if (!block.canBeBroken()) {
      return { success: false, reason: 'not_breakable' };
    }
    
    // 选择工具
    const tool = options?.tool || this.selectToolForBlock(block);
    if (!tool) {
      return { success: false, reason: 'no_tool' };
    }
    
    // 装备工具
    const inventory = this.player.getInventory();
    const toolSlot = inventory.findItem(tool);
    if (toolSlot === -1) {
      return { success: false, reason: 'no_tool' };
    }
    
    this.player.setHandSlot(toolSlot);
    
    // 挖掘
    const drops = await this.player.mineBlock(block);
    
    // 获取工具耐久
    const toolItem = inventory.getItem(toolSlot);
    const toolDurability = toolItem.getDurability();
    
    return {
      success: true,
      reason: 'success',
      blockName: block.name,
      drops,
      toolUsed: tool,
      toolDurability
    };
  }
  
  async placeBlock(x: number, y: number, z: number, blockName: string, facing?: string): Promise<PlaceBlockResult> {
    // 检查距离
    const distance = this.calculateDistance(this.player.pos, {x, y, z});
    if (distance > 5) {
      return { success: false, reason: 'too_far' };
    }
    
    // 检查位置是否被占用
    const block = mc.getBlock(x, y, z);
    if (block && !block.isAir()) {
      return { success: false, reason: 'position_occupied' };
    }
    
    // 查找方块
    const inventory = this.player.getInventory();
    const slot = inventory.findItem(blockName);
    if (slot === -1) {
      return { success: false, reason: 'no_block' };
    }
    
    // 装备方块
    this.player.setHandSlot(slot);
    
    // 放置
    const success = await this.player.placeBlock(x, y, z, facing);
    if (!success) {
      return { success: false, reason: 'cannot_place' };
    }
    
    return {
      success: true,
      reason: 'success',
      blockName,
      position: { x, y, z }
    };
  }
  
  async useBlock(x: number, y: number, z: number): Promise<UseBlockResult> {
    // 检查距离
    const distance = this.calculateDistance(this.player.pos, {x, y, z});
    if (distance > 5) {
      return { success: false, reason: 'too_far' };
    }
    
    // 获取方块
    const block = mc.getBlock(x, y, z);
    if (!block) {
      return { success: false, reason: 'block_not_found' };
    }
    
    // 使用方块
    const interactionType = await this.player.useBlock(block);
    
    return {
      success: true,
      reason: 'success',
      blockName: block.name,
      interactionType
    };
  }
  
  async areaOperation(mode: 'fill' | 'clear' | 'break' | 'vein', from: {x,y,z}, to: {x,y,z}, blockName?: string, radius: number = 5): Promise<AreaOperationResult> {
    // 计算区域大小
    const width = Math.abs(to.x - from.x) + 1;
    const height = Math.abs(to.y - from.y) + 1;
    const depth = Math.abs(to.z - from.z) + 1;
    const volume = width * height * depth;
    
    // 检查区域大小
    if (volume > 10000) {
      return { success: false, reason: 'too_large' };
    }
    
    if (mode === 'fill') {
      // 填充区域
      if (!blockName) {
        return { success: false, reason: 'no_block' };
      }
      const inventory = this.player.getInventory();
      const slot = inventory.findItem(blockName);
      if (slot === -1) {
        return { success: false, reason: 'no_block' };
      }
      this.player.setHandSlot(slot);
      
      let blocksPlaced = 0;
      for (let x = Math.min(from.x, to.x); x <= Math.max(from.x, to.x); x++) {
        for (let y = Math.min(from.y, to.y); y <= Math.max(from.y, to.y); y++) {
          for (let z = Math.min(from.z, to.z); z <= Math.max(from.z, to.z); z++) {
            await this.player.placeBlock(x, y, z);
            blocksPlaced++;
          }
        }
      }
      return { success: true, reason: 'success', blocksAffected: blocksPlaced, area: { width, height, depth } };
    }
    
    if (mode === 'clear') {
      // 清空区域
      let blocksCleared = 0;
      for (let x = Math.min(from.x, to.x); x <= Math.max(from.x, to.x); x++) {
        for (let y = Math.min(from.y, to.y); y <= Math.max(from.y, to.y); y++) {
          for (let z = Math.min(from.z, to.z); z <= Math.max(from.z, to.z); z++) {
            const block = mc.getBlock(x, y, z);
            if (block && !block.isAir()) {
              await this.player.breakBlock(block);
              blocksCleared++;
            }
          }
        }
      }
      return { success: true, reason: 'success', blocksAffected: blocksCleared, area: { width, height, depth } };
    }
    
    if (mode === 'break') {
      // 破坏区域内方块
      let blocksBroken = 0;
      const drops: Record<string, number> = {};
      for (let x = Math.min(from.x, to.x); x <= Math.max(from.x, to.x); x++) {
        for (let y = Math.min(from.y, to.y); y <= Math.max(from.y, to.y); y++) {
          for (let z = Math.min(from.z, to.z); z <= Math.max(from.z, to.z); z++) {
            const block = mc.getBlock(x, y, z);
            if (!block || block.isAir()) continue;
            const blockDrops = await this.player.mineBlock(block);
            blocksBroken++;
            for (const drop of blockDrops) {
              drops[drop.name] = (drops[drop.name] || 0) + drop.count;
            }
          }
        }
      }
      return {
        success: true, reason: 'success', blocksAffected: blocksBroken,
        drops: Object.entries(drops).map(([name, count]) => ({ name, count })),
        area: { width, height, depth }
      };
    }
    
    if (mode === 'vein') {
      // 连锁挖掘
      const startBlock = mc.getBlock(from.x, from.y, from.z);
      if (!startBlock || !this.isOre(startBlock)) {
        return { success: false, reason: 'no_ore' };
      }
      
      const oreName = startBlock.name;
      const visited = new Set<string>();
      const queue = [{x: from.x, y: from.y, z: from.z}];
      const oreBlocks: Array<{x,y,z}> = [];
      
      while (queue.length > 0) {
        const pos = queue.shift()!;
        const key = `${pos.x},${pos.y},${pos.z}`;
        if (visited.has(key)) continue;
        visited.add(key);
        
        const block = mc.getBlock(pos.x, pos.y, pos.z);
        if (!block || block.name !== oreName) continue;
        
        const distance = this.calculateDistance(from, pos);
        if (distance > radius) continue;
        
        oreBlocks.push(pos);
        
        const neighbors = [
          {x: pos.x+1, y: pos.y, z: pos.z}, {x: pos.x-1, y: pos.y, z: pos.z},
          {x: pos.x, y: pos.y+1, z: pos.z}, {x: pos.x, y: pos.y-1, z: pos.z},
          {x: pos.x, y: pos.y, z: pos.z+1}, {x: pos.x, y: pos.y, z: pos.z-1}
        ];
        
        for (const neighbor of neighbors) {
          if (!visited.has(`${neighbor.x},${neighbor.y},${neighbor.z}`)) {
            queue.push(neighbor);
          }
        }
      }
      
      if (oreBlocks.length === 0) {
        return { success: false, reason: 'no_ore' };
      }
      
      let blocksMined = 0;
      const drops: Record<string, number> = {};
      for (const pos of oreBlocks) {
        const block = mc.getBlock(pos.x, pos.y, pos.z);
        if (!block) continue;
        const blockDrops = await this.player.mineBlock(block);
        blocksMined++;
        for (const drop of blockDrops) {
          drops[drop.name] = (drops[drop.name] || 0) + drop.count;
        }
      }
      
      return {
        success: true, reason: 'success', blocksAffected: blocksMined,
        drops: Object.entries(drops).map(([name, count]) => ({ name, count }))
      };
    }
    
    return { success: false, reason: 'error' };
  }
  
  // 辅助方法
  private selectToolForBlock(block: Block): string | null {
    const blockName = block.name;
    
    // 根据方块类型选择工具
    if (blockName.includes('ore') || blockName === 'stone' || blockName === 'cobblestone') {
      return 'pickaxe';
    } else if (blockName === 'dirt' || blockName === 'grass_block' || blockName === 'sand') {
      return 'shovel';
    } else if (blockName.includes('log') || blockName.includes('planks')) {
      return 'axe';
    } else if (blockName === 'web') {
      return 'sword';
    }
    
    return null;
  }
  
  private isOre(block: Block): boolean {
    return block.name.includes('ore');
  }
}
```

## 5. 总结

### 5.1 核心设计

| 组件 | 职责 |
|---|---|
| **LLM（大脑）** | 决定要挖/放/用方块，调用方块工具 |
| **方块工具** | 封装方块操作，返回执行结果 |
| **执行AI（假人）** | 负责具体方块操作（自动选择工具、批量操作等） |

### 5.2 关键特性

1. **自动选择工具**：执行AI自动选择合适工具（挖石头用镐、挖土用锹）
2. **批量操作**：支持区域填充、批量破坏、矿脉挖掘
3. **有反馈**：工具返回操作结果（成功/失败/详情）
4. **可失败**：工具损坏、方块不可破坏、距离太远等

### 5.3 工具列表

| 工具 | 功能 |
|---|---|
| mine_block | 挖掘方块 |
| place_block | 放置方块 |
| use_block | 右键使用方块 |
| area_operation | 区域操作（填充/清空/破坏/连锁挖掘） |
| collect_block | 自动采集方块（支持数量控制和预算时间） |
