# 生存与状态工具设计（Survival & Status Tools Design）

> 版本：v1.0
> 日期：2026-07-01
> 架构：LLM 调用工具 → 执行AI 执行操作 → 返回结果
> 协议：遵循 `00-工具协议规范.md` 统一接口契约

---

## 0. 设计原则

### 0.1 核心概念

**生存操作由执行AI负责，LLM 只调用工具**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 决定要吃/睡/使用物品                               │
│  - 调用生存工具                                       │
└────────────────┬────────────────────────────────────┘
                 │ 调用工具
                 ▼
        ┌────────────────┐
        │  生存工具        │
        │  - eat          │
        │  - sleep        │
        │  - use_item     │
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 选择食物     │
        │  - 放置床       │
        │  - 使用物品     │
        └────────┬───────┘
                 │ 返回结果
                 ▼
        ┌────────────────┐
        │  工具返回        │
        │  - 成功/失败    │
        │  - 恢复效果     │
        └────────────────┘
```

### 0.2 设计原则

| 原则 | 说明 |
|---|---|
| **一键操作** | 吃/睡等操作尽量一键完成 |
| **有反馈** | 工具返回操作结果（成功/失败/恢复效果） |
| **可失败** | 没有食物、没有床、天不是黑等 |
| **简单直接** | 工具粒度适中，避免过度设计 |

## 2. 工具定义

### 2.1 eat（吃东西）

**参数规范（ParameterSchema）**：

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|:----:|------|:------:|
| `food_name` | `string` | 否 | 食物名称（可选，不填则自动选择最佳食物） | - |

**返回数据（`EatData`）**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `foodUsed` | `string` | 实际使用的食物名称 |
| `hungerRestored` | `number` | 恢复的饥饿值 |
| `saturationRestored` | `number` | 恢复的饱和度 |

**返回格式**：`ResultEnvelope<EatData>`

**错误码**：

| 错误码 | 含义 |
|--------|------|
| `NO_FOOD` | 背包中没有食物 |
| `NOT_HUNGRY` | 不饿，无需进食 |
| `INTERNAL_ERROR` | 内部错误 |

**示例调用**：

```
LLM 调用: eat()
返回:
{
  "success": true,
  "data": {
    "foodUsed": "cooked_beef",
    "hungerRestored": 8,
    "saturationRestored": 12.8
  },
  "meta": {
    "duration": 500,
    "cost": {}
  }
}
```

**自然语言返回**：

```
吃东西成功: 熟牛肉
恢复饥饿: 8 恢复饱和度: 12.8
```

### 2.2 sleep（睡觉/起床）

**参数规范（ParameterSchema）**：

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|:----:|------|:------:|
| `action` | `string` | 是 | 睡觉或起床，枚举值：`sleep`、`wake` | - |
| `bed_pos` | `object` | 否 | 床的位置坐标 `{x, y, z}`（不填则自动寻找或放置） | - |
| `wait_seconds` | `number` | 否 | 等待时间（秒），不填则等待自然醒来 | - |

**返回数据（`SleepData`）**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `bedPosition` | `object` | 床的位置坐标（可选） |
| `timeSkipped` | `number` | 跳过的时间（ticks，可选） |

**返回格式**：`ResultEnvelope<SleepData>`

**错误码**：

| 错误码 | 含义 |
|--------|------|
| `NOT_FOUND` | 找不到床 / 当前未在睡觉状态 |
| `NOT_NIGHT` | 还不是夜晚，无法睡觉 |
| `MONSTERS_NEARBY` | 附近有怪物，无法睡觉 |
| `BED_OCCUPIED` | 床被占用 |
| `INTERNAL_ERROR` | 内部错误 |

**示例调用**：

```
LLM 调用: sleep(action="sleep")
返回:
{
  "success": true,
  "data": {
    "bedPosition": { "x": 128, "y": 64, "z": -256 },
    "timeSkipped": 10000
  },
  "meta": {
    "duration": 1000,
    "cost": {}
  }
}
```

**自然语言返回**：

```
睡觉成功: 床位置 (128, 64, -256)
跳过时间: 10000 ticks
```

```
LLM 调用: sleep(action="wake")
返回:
{
  "success": true,
  "data": {},
  "meta": {
    "duration": 100,
    "cost": {}
  }
}
```

**自然语言返回**：

```
起床成功
```

### 2.3 use_item（使用物品）

**参数规范（ParameterSchema）**：

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|:----:|------|:------:|
| `item_name` | `string` | 是 | 物品名称 | - |
| `mode` | `string` | 是 | 使用模式，枚举值：`use`（使用）、`drink`（喝药水）、`throw`（投掷） | - |
| `target` | `object` | 否 | 目标位置 `{x, y, z}`（投掷模式需要） | - |

**返回数据（`UseItemData`）**：

| 字段 | 类型 | 说明 |
|------|------|------|
| `itemUsed` | `string` | 实际使用的物品名称 |
| `effect` | `string` | 使用效果描述（可选） |
| `targetHit` | `string` | 击中的目标（投掷模式，可选） |

**返回格式**：`ResultEnvelope<UseItemData>`

**错误码**：

| 错误码 | 含义 |
|--------|------|
| `ITEM_NOT_FOUND` | 背包中找不到指定物品 |
| `INVALID_PARAMS` | 物品不可使用 / 不可投掷 |

### 3.2 执行AI接口

```typescript
// 执行AI接口（生存操作）
interface ExecutionAI {
  // 饮食
  eat(foodName?: string): Promise<ResultEnvelope<EatData>>;
  
  // 睡眠
  sleep(action: 'sleep' | 'wake', bedPos?: {x, y, z}, waitSeconds?: number): Promise<ResultEnvelope<SleepData>>;
  
  // 物品使用
  useItem(itemName: string, mode: 'use' | 'drink' | 'throw', target?: {x, y, z}): Promise<ResultEnvelope<UseItemData>>;
}
```

### 3.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  
  constructor(player: Player) {
    this.player = player;
  }
  
  async eat(foodName?: string): Promise<ResultEnvelope<EatData>> {
    const inventory = this.player.getInventory();
    
    // 检查是否饥饿
    if (this.player.getHung() >= 20) {
      return { success: false, error: { code: 'NOT_HUNGRY', message: '玩家不饿，无需进食' }, meta: { duration: 0 } };
    }
    
    let food: Item | null = null;
    
    if (foodName) {
      // 使用指定食物
      const slot = inventory.findItem(foodName);
      if (slot === -1) {
        return { success: false, error: { code: 'NO_FOOD', message: '背包中没有食物: ' + foodName }, meta: { duration: 0 } };
      }
      food = inventory.getItem(slot);
    } else {
      // 自动选择最佳食物
      food = this.findBestFood(inventory);
      if (!food) {
        return { success: false, error: { code: 'NO_FOOD', message: '背包中没有可用食物' }, meta: { duration: 0 } };
      }
    }
    
    // 吃东西
    const slot = inventory.findItem(food.name);
    this.player.eat(slot);
    
    // 计算恢复效果
    const hungerRestored = this.getFoodHunger(food.name);
    const saturationRestored = this.getFoodSaturation(food.name);
    
    return {
      success: true,
      data: {
        foodUsed: food.name,
        hungerRestored,
        saturationRestored
      },
      meta: { duration: 500, cost: {} }
    };
  }
  
  async sleep(action: 'sleep' | 'wake', bedPos?: {x: number; y: number; z: number}, waitSeconds?: number): Promise<ResultEnvelope<SleepData>> {
    if (action === 'wake') {
      if (!this.player.isSleeping()) {
        return { success: false, error: { code: 'NOT_FOUND', message: '当前未在睡觉状态' }, meta: { duration: 0 } };
      }
      this.player.wakeUp();
      return { success: true, data: {}, meta: { duration: 100, cost: {} } };
    }
    
    // 检查是否是夜晚
    const time = mc.getTime();
    if (time < 12541 || time > 23458) {
      return { success: false, error: { code: 'NOT_NIGHT', message: '还不是夜晚，无法睡觉' }, meta: { duration: 0 } };
    }
    
    // 检查附近是否有怪物
    const nearbyHostile = this.findNearbyHostile(8);
    if (nearbyHostile) {
      return { success: false, error: { code: 'MONSTERS_NEARBY', message: '附近有怪物，无法睡觉' }, meta: { duration: 0 } };
    }
    
    let bedBlock: Block | null = null;
    
    if (bedPos) {
      // 使用指定床
      bedBlock = mc.getBlock(bedPos.x, bedPos.y, bedPos.z);
      if (!bedBlock || !this.isBed(bedBlock)) {
        return { success: false, error: { code: 'NOT_FOUND', message: '找不到床' }, meta: { duration: 0 } };
      }
    } else {
      // 自动寻找床
      bedBlock = this.findNearestBed();
      if (!bedBlock) {
        // 尝试放置床
        const placed = await this.placeBed();
        if (!placed) {
          return { success: false, error: { code: 'NOT_FOUND', message: '找不到床，也无法放置床' }, meta: { duration: 0 } };
        }
        bedBlock = placed;
      }
    }
    
    // 检查床是否被占用
    if (this.isBedOccupied(bedBlock)) {
      return { success: false, error: { code: 'BED_OCCUPIED', message: '床被占用' }, meta: { duration: 0 } };
    }
    
    // 走到床边
    await this.moveTo(bedBlock.pos.x, bedBlock.pos.y, bedBlock.pos.z);
    
    // 睡觉
    this.player.sleep(bedBlock);
    
    // 等待醒来（使用指定等待时间或自然醒来）
    if (waitSeconds) {
      await this.delay(waitSeconds * 1000);
      this.player.wakeUp();
    } else {
      await this.waitForWakeUp();
    }
    
    return {
      success: true,
      data: {
        bedPosition: bedBlock.pos,
        timeSkipped: 10000  // 大约跳过的时间
      },
      meta: { duration: 1000, cost: {} }
    };
  }
  
  async useItem(itemName: string, mode: 'use' | 'drink' | 'throw', target?: {x: number; y: number; z: number}): Promise<ResultEnvelope<UseItemData>> {
    const inventory = this.player.getInventory();
    const slot = inventory.findItem(itemName);
    
    if (slot === -1) {
      return { success: false, error: { code: 'ITEM_NOT_FOUND', message: '背包中找不到物品: ' + itemName }, meta: { duration: 0 } };
    }
    
    const item = inventory.getItem(slot);
    
    if (mode === 'drink') {
      // 喝药水
      this.player.useItem(slot);
      const effects = this.getPotionEffects(item);
      return {
        success: true,
        data: {
          itemUsed: item.name,
          effect: effects.map(e => `${e.name} ${e.level}`).join(', ')
        },
        meta: { duration: 300, cost: {} }
      };
    }
    
    if (mode === 'throw') {
      // 投掷物品
      if (!this.isThrowable(item)) {
        return { success: false, error: { code: 'INVALID_PARAMS', message: '该物品不可投掷' }, meta: { duration: 0 } };
      }
      let targetHit: string | undefined;
      if (target) {
        this.player.throwItem(slot, target);
        targetHit = `(${target.x}, ${target.y}, ${target.z})`;
      } else {
        this.player.throwItem(slot);
      }
      return {
        success: true,
        data: {
          itemUsed: item.name,
          targetHit
        },
        meta: { duration: 500, cost: {} }
      };
    }
    
    // 默认使用模式
    if (!this.isUsable(item)) {
      return { success: false, error: { code: 'INVALID_PARAMS', message: '该物品不可使用' }, meta: { duration: 0 } };
    }
    
    this.player.useItem(slot);
    const effect = this.getItemUseEffect(item);
    
    return {
      success: true,
      data: {
        itemUsed: item.name,
        effect
      },
      meta: { duration: 300, cost: {} }
    };
  }
```

## 5. 总结

### 5.1 核心设计

| 组件 | 职责 |
|---|---|
| **LLM（大脑）** | 决定要吃/睡/使用物品，调用生存工具 |
| **生存工具** | 封装生存操作，返回执行结果 |
| **执行AI（假人）** | 负责具体生存操作（选择食物、放置床、使用物品等） |

### 5.2 关键特性

1. **一键操作**：eat/sleep 一键完成
2. **自动选择**：自动选择最佳食物/药水
3. **有反馈**：工具返回操作结果（成功/失败/恢复效果）
4. **可失败**：没有食物、没有床、天不是黑等

### 5.3 工具列表

| 工具 | 功能 |
|---|---|
| eat | 吃东西（自动选择最佳食物） |
| sleep | 睡觉/起床 |
| use_item | 使用物品（使用/喝药水/投掷） |
