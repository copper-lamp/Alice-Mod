# 生存与状态工具设计（Survival & Status Tools Design）

> 版本：v1.0
> 日期：2026-07-01
> 架构：LLM 调用工具 → 执行AI 执行操作 → 返回结果

---

## 0. 设计原则

### 0.1 核心概念

**生存操作由执行AI负责，LLM 只调用工具**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 决定要吃/睡/使用物品                               │
│  - 调用生存工具                                       │
└────────────────┬────────────────────────────────────┘
                 │ 调用工具
                 ▼
        ┌────────────────┐
        │  生存工具        │
        │  - eat          │
        │  - sleep        │
        │  - use_item     │
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 选择食物     │
        │  - 放置床       │
        │  - 使用物品     │
        └────────┬───────┘
                 │ 返回结果
                 ▼
        ┌────────────────┐
        │  工具返回        │
        │  - 成功/失败    │
        │  - 恢复效果     │
        └────────────────┘
```

### 0.2 设计原则

| 原则 | 说明 |
|---|---|
| **一键操作** | 吃/睡等操作尽量一键完成 |
| **有反馈** | 工具返回操作结果（成功/失败/恢复效果） |
| **可失败** | 没有食物、没有床、天不是黑等 |
| **简单直接** | 工具粒度适中，避免过度设计 |

### 3.2 执行AI接口

```typescript
// 执行AI接口（生存操作）
interface ExecutionAI {
  // 饮食
  eat(foodName?: string): Promise<EatResult>;
  
  // 睡眠
  sleep(action: 'sleep' | 'wake', bedPos?: {x, y, z}, waitSeconds?: number): Promise<SleepResult>;
  
  // 物品使用
  useItem(itemName: string, mode: 'use' | 'drink' | 'throw', target?: {x, y, z}): Promise<UseItemResult>;
}
```

### 3.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  
  constructor(player: Player) {
    this.player = player;
  }
  
  async eat(foodName?: string): Promise<EatResult> {
    const inventory = this.player.getInventory();
    
    // 检查是否饥饿
    if (this.player.getHung() >= 20) {
      return { success: false, reason: 'not_hungry' };
    }
    
    let food: Item | null = null;
    
    if (foodName) {
      // 使用指定食物
      const slot = inventory.findItem(foodName);
      if (slot === -1) {
        return { success: false, reason: 'no_food' };
      }
      food = inventory.getItem(slot);
    } else {
      // 自动选择最佳食物
      food = this.findBestFood(inventory);
      if (!food) {
        return { success: false, reason: 'no_food' };
      }
    }
    
    // 吃东西
    const slot = inventory.findItem(food.name);
    this.player.eat(slot);
    
    // 计算恢复效果
    const hungerRestored = this.getFoodHunger(food.name);
    const saturationRestored = this.getFoodSaturation(food.name);
    
    return {
      success: true,
      reason: 'success',
      foodUsed: food.name,
      hungerRestored,
      saturationRestored
    };
  }
  
  async sleep(action: 'sleep' | 'wake', bedPos?: {x: number; y: number; z: number}, waitSeconds?: number): Promise<SleepResult> {
    if (action === 'wake') {
      if (!this.player.isSleeping()) {
        return { success: false, reason: 'not_sleeping' };
      }
      this.player.wakeUp();
      return { success: true, reason: 'success' };
    }
    
    // 检查是否是夜晚
    const time = mc.getTime();
    if (time < 12541 || time > 23458) {
      return { success: false, reason: 'not_night' };
    }
    
    // 检查附近是否有怪物
    const nearbyHostile = this.findNearbyHostile(8);
    if (nearbyHostile) {
      return { success: false, reason: 'monsters_nearby' };
    }
    
    let bedBlock: Block | null = null;
    
    if (bedPos) {
      // 使用指定床
      bedBlock = mc.getBlock(bedPos.x, bedPos.y, bedPos.z);
      if (!bedBlock || !this.isBed(bedBlock)) {
        return { success: false, reason: 'no_bed' };
      }
    } else {
      // 自动寻找床
      bedBlock = this.findNearestBed();
      if (!bedBlock) {
        // 尝试放置床
        const placed = await this.placeBed();
        if (!placed) {
          return { success: false, reason: 'no_bed' };
        }
        bedBlock = placed;
      }
    }
    
    // 检查床是否被占用
    if (this.isBedOccupied(bedBlock)) {
      return { success: false, reason: 'bed_occupied' };
    }
    
    // 走到床边
    await this.moveTo(bedBlock.pos.x, bedBlock.pos.y, bedBlock.pos.z);
    
    // 睡觉
    this.player.sleep(bedBlock);
    
    // 等待醒来（使用指定等待时间或自然醒来）
    if (waitSeconds) {
      await this.delay(waitSeconds * 1000);
      this.player.wakeUp();
    } else {
      await this.waitForWakeUp();
    }
    
    return {
      success: true,
      reason: 'success',
      bedPosition: bedBlock.pos,
      timeSkipped: 10000  // 大约跳过的时间
    };
  }
  
  async useItem(itemName: string, mode: 'use' | 'drink' | 'throw', target?: {x: number; y: number; z: number}): Promise<UseItemResult> {
    const inventory = this.player.getInventory();
    const slot = inventory.findItem(itemName);
    
    if (slot === -1) {
      return { success: false, reason: 'not_found' };
    }
    
    const item = inventory.getItem(slot);
    
    if (mode === 'drink') {
      // 喝药水
      this.player.useItem(slot);
      const effects = this.getPotionEffects(item);
      return {
        success: true,
        reason: 'success',
        itemUsed: item.name,
        effect: effects.map(e => `${e.name} ${e.level}`).join(', ')
      };
    }
    
    if (mode === 'throw') {
      // 投掷物品
      if (!this.isThrowable(item)) {
        return { success: false, reason: 'not_throwable' };
      }
      let targetHit: string | undefined;
      if (target) {
        this.player.throwItem(slot, target);
        targetHit = `(${target.x}, ${target.y}, ${target.z})`;
      } else {
        this.player.throwItem(slot);
      }
      return {
        success: true,
        reason: 'success',
        itemUsed: item.name,
        targetHit
      };
    }
    
    // 默认使用模式
    if (!this.isUsable(item)) {
      return { success: false, reason: 'not_usable' };
    }
    
    this.player.useItem(slot);
    const effect = this.getItemUseEffect(item);
    
    return {
      success: true,
      reason: 'success',
      itemUsed: item.name,
      effect
    };
  }
```

## 5. 总结

### 5.1 核心设计

| 组件 | 职责 |
|---|---|
| **LLM（大脑）** | 决定要吃/睡/使用物品，调用生存工具 |
| **生存工具** | 封装生存操作，返回执行结果 |
| **执行AI（假人）** | 负责具体生存操作（选择食物、放置床、使用物品等） |

### 5.2 关键特性

1. **一键操作**：eat/sleep 一键完成
2. **自动选择**：自动选择最佳食物/药水
3. **有反馈**：工具返回操作结果（成功/失败/恢复效果）
4. **可失败**：没有食物、没有床、天不是黑等

### 5.3 工具列表

| 工具 | 功能 |
|---|---|
| eat | 吃东西（自动选择最佳食物） |
| sleep | 睡觉/起床 |
| use_item | 使用物品（使用/喝药水/投掷） |
