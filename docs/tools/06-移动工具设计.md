# 移动工具设计（Movement Tools Design）

> 版本：v2.0
> 日期：2026-07-04
> 架构：LLM 调用工具 → 执行AI（假人）执行移动 → 返回结果
> 变更：工具精简为 3 个（move_to/ride/dismount），move_to 通过 target_type 区分目标类型，移除方向移动工具

---

## 0. 设计原则

### 0.1 核心概念

**移动由执行AI负责，LLM 只调用基础移动和骑乘工具**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 决定要去哪里                                       │
│  - 调用基础移动工具                                   │
└────────────────┬────────────────────────────────────┘
                 │ 调用工具
                 ▼
        ┌────────────────┐
        │  移动工具        │
        │  - move_to      │
        │  - ride/dismount│
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 寻路算法     │
        │  - 避障         │
        │  - 跳跃/攀爬    │
        │  - 游泳         │
        │  - 视角控制     │
        │  （自动处理）    │
        └────────┬───────┘
                 │ 返回结果
                 ▼
        ┌────────────────┐
        │  工具返回        │
        │  - 成功/失败    │
        │  - 最终位置     │
        │  - 耗时/消耗    │
        └────────────────┘
```

### 0.2 设计原则

| 原则 | 说明 |
|---|---|
| **粒度适中** | 一个工具 = 一个"动作"，不是"脚本" |
| **有反馈** | 工具返回执行结果（成功/失败/中断原因） |
| **可失败** | 寻路失败、被阻挡、体力不足等 |
| **有成本** | 时间、饥饿、耐久 |
| **可中断** | 移动过程中可被事件中断（遇敌、受伤） |

## 2. 工具定义

### 2.1 move_to（移动到目标位置）

```typescript
interface MoveToParams {
  target: 
    | { x: number; y: number; z: number }   // 坐标目标
    | { entity_id: string };                  // 实体目标
  target_type: 'coordinate' | 'entity' | 'block';
  distance?: number;     // 目标距离（默认 2 格，仅 entity/block 有效）
  sprint?: boolean;      // 是否疾跑（默认 false）
  options?: {
    timeout?: number;      // 超时时间（毫秒，默认 30000）
    avoidHostile?: boolean; // 避开敌对生物（默认 false）
    allowBreak?: boolean;   // 允许破坏方块（默认 false）
    allowPlace?: boolean;   // 允许放置方块（默认 false）
    pathfinding?: 'astar' | 'dijkstra';  // 寻路算法（默认 astar）
  };
}

interface MoveToResult {
  success: boolean;
  reason?: 'success' | 'timeout' | 'blocked' | 'no_path' | 'entity_moved' | 'entity_not_found' | 'interrupted' | 'error';
  finalPosition?: { x: number; y: number; z: number };
  finalDistance?: number;  // 最终与目标的距离
  distance?: number;       // 实际移动距离
  duration?: number;       // 耗时（毫秒）
  hungerCost?: number;     // 饥饿消耗
  errorMessage?: string;
}
```

**示例调用**（坐标目标）：

```
LLM 调用: move_to(target={x: 128, y: 64, z: -256}, target_type='coordinate')
返回:
{
  "success": true,
  "reason": "success",
  "finalPosition": { "x": 128.5, "y": 64, "z": -256.2 },
  "distance": 15.3,
  "duration": 4500,
  "hungerCost": 0.5
}
```

```

  "success": true,
  "reason": "success",

  "finalDistance": 2.8,
```

**自然语言返回**：

## 3. 工具实现

### 3.1 工具注册

```typescript
// 注册移动工具
function registerMovementTools() {
  // move_to
  registerTool('move_to', {
    description: '移动到目标位置（支持坐标、实体、方块目标，执行AI负责寻路和避障）',
    parameters: {
      target: { type: 'object', required: true },
      target_type: { type: 'string', enum: ['coordinate', 'entity', 'block'], required: true },
      distance: { type: 'number', default: 2 },
      sprint: { type: 'boolean', default: false },
      options: { type: 'object', optional: true }
    },
    execute: async (params) => {
      const { target, target_type, distance = 2, sprint = false, options = {} } = params;
      
      let targetPos;
      
      if (target_type === 'coordinate' || target_type === 'block') {
        targetPos = { x: target.x, y: target.y, z: target.z };
      } else if (target_type === 'entity') {
        const entity = getEntityById(target.entity_id);
        if (!entity) {
          return { success: false, reason: 'entity_not_found' };
        }
        targetPos = { x: entity.pos.x, y: entity.pos.y, z: entity.pos.z };
      }
      
      // 委托给执行AI
      const result = await executionAI.moveTo(targetPos.x, targetPos.y, targetPos.z, {
        ...options,
        targetDistance: distance,
        sprint
      });
      
      return result;
    }
  });
  
  // ride
  registerTool('ride', {
    description: '骑乘实体（马、猪、船等）',
    parameters: {
      entity_id: { type: 'string', required: true }
    },
    execute: async (params) => {
      return await executionAI.ride(params.entity_id);
    }
  });
  
  // dismount
  registerTool('dismount', {
    description: '下马',
    parameters: {},
    execute: async () => {
      return await executionAI.dismount();
    }
  });
}
```

### 3.2 执行AI接口

```typescript
// 执行AI接口（假人控制）
interface ExecutionAI {
  // 基础移动
  moveTo(x: number, y: number, z: number, options?: MoveOptions): Promise<MoveToResult>;
  
  // 骑乘
  ride(entity_id: string): Promise<RideResult>;
  dismount(): Promise<DismountResult>;
}

// 移动选项
interface MoveOptions {
  timeout?: number;
  avoidHostile?: boolean;
  allowBreak?: boolean;
  allowPlace?: boolean;
  sprint?: boolean;         // 是否疾跑
  pathfinding?: 'astar' | 'dijkstra';
  targetDistance?: number;  // 目标距离（接近实体/方块时使用）
}
```

### 3.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  private pathfinder: Pathfinder;
  
  constructor(player: Player) {
    this.player = player;
    this.pathfinder = new Pathfinder();
  }
  
  async moveTo(x: number, y: number, z: number, options?: MoveOptions): Promise<MoveToResult> {
    const startTime = Date.now();
    const startPos = this.player.pos;
    
    try {
      // 1. 计算路径
      const path = await this.pathfinder.findPath(
        startPos,
        { x, y, z },
        {
          avoidHostile: options?.avoidHostile,
          allowBreak: options?.allowBreak,
          allowPlace: options?.allowPlace
        }
      );
      
      if (!path) {
        return {
          success: false,
          reason: 'no_path',
          finalPosition: startPos,
          duration: Date.now() - startTime
        };
      }
      
      // 2. 执行移动（自动处理跳跃、攀爬、游泳等）
      const timeout = options?.timeout || 30000;
      const result = await this.executePath(path, timeout);
      
      if (!result.success) {
        return {
          success: false,
          reason: result.reason,
          finalPosition: this.player.pos,
          distance: calculateDistance(startPos, this.player.pos),
          duration: Date.now() - startTime,
          hungerCost: result.hungerCost
        };
      }
      
      // 3. 返回结果
      return {
        success: true,
        reason: 'success',
        finalPosition: this.player.pos,
        distance: calculateDistance(startPos, this.player.pos),
        duration: Date.now() - startTime,
        hungerCost: result.hungerCost
      };
      
    } catch (error) {
      return {
        success: false,
        reason: 'error',
        finalPosition: this.player.pos,
        duration: Date.now() - startTime,
        errorMessage: error.message
      };
    }
  }
  
  private async executePath(path: Path, timeout: number): Promise<{success: boolean; reason?: string; hungerCost?: number}> {
    const startTime = Date.now();
    let hungerCost = 0;
    
    for (const node of path.nodes) {
      // 检查超时
      if (Date.now() - startTime > timeout) {
        return { success: false, reason: 'timeout', hungerCost };
      }
      
      // 移动到下一个节点（执行AI自动处理跳跃、攀爬、游泳等）
      const moveResult = await this.moveToNode(node);
      if (!moveResult.success) {
        return { success: false, reason: moveResult.reason, hungerCost };
      }
      
      // 累计饥饿消耗
      hungerCost += moveResult.hungerCost;
    }
    
    return { success: true, hungerCost };
  }
  
  private async moveToNode(node: PathNode): Promise<{success: boolean; reason?: string; hungerCost: number}> {
    // 执行AI自动处理：
    // - 走路
    // - 跳跃（遇到障碍）
    // - 攀爬（遇到梯子/藤蔓）
    // - 游泳（遇到水）
    // - 视角控制（自动看向目标）
    
    // 示例：走路
    const dx = node.x - this.player.pos.x;
    const dz = node.z - this.player.pos.z;
    const distance = Math.sqrt(dx * dx + dz * dz);
    
    // 设置朝向（自动）
    const yaw = Math.atan2(dz, dx) * 180 / Math.PI;
    this.player.setRotation(yaw, 0);
    
    // 开始移动
    this.player.setVelocity({ x: dx / distance * 0.2, y: 0, z: dz / distance * 0.2 });
    
    // 等待到达
    await this.waitForPosition(node, 1000);
    
    // 计算饥饿消耗
    const hungerCost = distance * 0.01;  // 每格 0.01 饥饿
    
    return { success: true, hungerCost };
  }
  
  async ride(entity_id: string): Promise<RideResult> {
    const entity = getEntityById(entity_id);
    if (!entity) {
      return { success: false, reason: 'entity_not_found' };
    }
    
    // 检查是否可骑乘
    if (!this.isRideable(entity)) {
      return { success: false, reason: 'not_rideable' };
    }
    
    // 检查距离
    const distance = calculateDistance(this.player.pos, entity.pos);
    if (distance > 3) {
      return { success: false, reason: 'too_far' };
    }
    
    // 执行骑乘
    this.player.ride(entity);
    
    return {
      success: true,
      reason: 'success',
      isRiding: true,
      mountType: entity.type
    };
  }
  
  async dismount(): Promise<DismountResult> {
    if (!this.player.isRiding()) {
      return { success: false, reason: 'not_riding' };
    }
    
    this.player.dismount();
    
    return {
      success: true,
      reason: 'success',
      isRiding: false
    };
  }
  
  // ... 其他方法
}
```

## 5. 总结

### 5.1 核心设计

| 组件 | 职责 |
|---|---|
| **LLM（大脑）** | 决定要去哪里，调用基础移动工具 |
| **移动工具** | 封装移动能力，返回执行结果 |
| **执行AI（假人）** | 负责具体移动操作（寻路、避障、跳跃、攀爬、游泳、视角等） |

### 5.2 关键特性

1. **工具粒度适中**：一个工具 = 一个"动作"（move_to, ride, dismount 等）
2. **有反馈**：工具返回执行结果（成功/失败/中断原因）
3. **可失败**：寻路失败、超时、被阻挡等
4. **有成本**：时间、饥饿
5. **可中断**：移动过程中可被事件中断
6. **执行AI自动处理**：跳跃、攀爬、游泳、视角控制等由执行AI自动处理

### 5.3 工具列表

| 类别 | 工具 |
|---|---|
| 基础移动 | move_to（通过 target_type 区分坐标/实体/方块目标） |
| 骑乘 | ride, dismount |
