# 移动工具设计（Movement Tools Design）

> 版本：v2.0
> 日期：2026-07-04
> 架构：LLM 调用工具 → 执行AI（假人）执行移动 → 返回结果
> 协议：遵循 `00-工具协议规范.md` 统一接口契约
> 变更：工具精简为 3 个（move\_to/ride/dismount），move\_to 通过 target\_type 区分目标类型，移除方向移动工具

***

## 0. 设计原则

### 0.1 核心概念

**移动由执行AI负责，LLM 只调用基础移动和骑乘工具**

```
┌─────────────────────────────────────────────────────┐
│  LLM（大脑）                                         │
│  - 决定要去哪里                                       │
│  - 调用基础移动工具                                   │
└────────────────┬────────────────────────────────────┘
                 │ 调用工具
                 ▼
        ┌────────────────┐
        │  移动工具        │
        │  - move_to      │
        │  - ride/dismount│
        └────────┬───────┘
                 │ 委托执行
                 ▼
        ┌────────────────┐
        │  执行AI（假人）  │
        │  - 寻路算法     │
        │  - 避障         │
        │  - 跳跃/攀爬    │
        │  - 游泳         │
        │  - 视角控制     │
        │  （自动处理）    │
        └────────┬───────┘
                 │ 返回结果
                 ▼
        ┌────────────────┐
        │  工具返回        │
        │  - 成功/失败    │
        │  - 最终位置     │
        │  - 耗时/消耗    │
        └────────────────┘
```

### 0.2 设计原则

| 原则       | 说明                   |
| -------- | -------------------- |
| **粒度适中** | 一个工具 = 一个"动作"，不是"脚本" |
| **有反馈**  | 工具返回执行结果（成功/失败/中断原因） |
| **可失败**  | 寻路失败、被阻挡、体力不足等       |
| **有成本**  | 时间、饥饿、耐久             |
| **可中断**  | 移动过程中可被事件中断（遇敌、受伤）   |

## 2. 工具定义

### 2.1 move\_to（移动到目标位置）

**ParameterSchema**（参考 [00-工具协议规范.md §2.2](00-工具协议规范.md#22-参数规范parameterschema)）：

| 参数名           | 类型        | 必填 | 默认值     | 描述                                                                                                           |
| ------------- | --------- | -- | ------- | ------------------------------------------------------------------------------------------------------------ |
| `target`      | `object`  | 是  | —       | 目标：`{x, y, z}` 坐标或 `{entity_id}` 实体引用                                                                        |
| `target_type` | `string`  | 是  | —       | 目标类型：`'coordinate'` / `'entity'` / `'block'`                                                                 |
| `distance`    | `number`  | 否  | `5000`  | 停止距离（格），仅 entity/block 有效                                                                                    |
| `sprint`      | `boolean` | 否  | `false` | 是否疾跑                                                                                                         |
| `options`     | `object`  | 否  | —       | 可选配置，含 `timeout`（超时，默认300000ms）、`avoidHostile`（避开敌对）、`allowBreak`（允许破坏方块）、`allowPlace`（允许放置方块）、`pathfinding` |

**ResultEnvelope\<MoveToData>**（参考 [00-工具协议规范.md §3](00-工具协议规范.md#3-返回信封resultenvelope)）：

```typescript
interface MoveToData {
  finalPosition: { x: number; y: number; z: number };
  finalDistance?: number;  // 最终与目标的距离
  distance?: number;       // 实际移动距离
  hungerCost?: number;     // 饥饿消耗
}
```

**ErrorCode**（参考 [00-工具协议规范.md §4.2](00-工具协议规范.md#42-领域错误码)）：

| 错误码              | 含义      | 触发条件    |
| ---------------- | ------- | ------- |
| `TIMEOUT`        | 操作超时    | 寻路超时    |
| `BLOCKED`        | 路径被阻挡   | 无法通过    |
| `NO_PATH`        | 无法找到路径  | 无可达路径   |
| `ENTITY_MOVED`   | 目标实体已移动 | 实体目标已移动 |
| `NOT_FOUND`      | 目标不存在   | 实体目标不存在 |
| `CANCELLED`      | 操作被取消   | 被事件中断   |
| `INTERNAL_ERROR` | 内部错误    | 未预期异常   |

**示例调用**（坐标目标）：

```
LLM 调用: move_to(target={x: 128, y: 64, z: -256}, target_type='coordinate')
返回:
{
  "success": true,
  "data": {
    "finalPosition": { "x": 128.5, "y": 64, "z": -256.2 },
    "distance": 15.3
  },
  "meta": {
    "duration": 4500,
    "cost": { "hunger": 0.5 }
  }
}
```

**示例调用**（实体目标）：

```
LLM 调用: move_to(target={entity_id: "entity_123"}, target_type='entity', distance=3)
返回:
{
  "success": true,
  "data": {
    "finalPosition": { "x": 130.2, "y": 64, "z": -258.5 },
    "finalDistance": 2.8,
    "distance": 12.5
  },
  "meta": {
    "duration": 2000,
    "cost": { "hunger": 0.3 }
  }
}
```

**自然语言返回**：

```

距离: 15.3格 耗时: 4.5秒 饥饿消耗: 0.5
```

**失败示例**：

```

返回:
{

  "error": {
    "code": "TIMEOUT",
    "message": "移动超时，已耗时 30000ms"
```

## 3. 工具实现

### 3.1 工具注册

```typescript
// 注册移动工具
function registerMovementTools() {
  // move_to
  ToolRegistry.register({
    name: 'move_to',
    description: '移动到目标位置（支持坐标、实体、方块目标，执行AI负责寻路和避障）',
    parameterSchema: {
      type: 'object',
      properties: {
        target: { type: 'object', description: '目标：{x,y,z} 坐标或 {entity_id} 实体引用' },
        target_type: { type: 'string', enum: ['coordinate', 'entity', 'block'], description: '目标类型' },
        distance: { type: 'number', description: '停止距离（格）', default: 2 },
        sprint: { type: 'boolean', description: '是否疾跑', default: false },
        options: { type: 'object', description: '可选配置' }
      },
      required: ['target', 'target_type']
    },
    async invoke(params) {
      const { target, target_type, distance = 2, sprint = false, options = {} } = params;
      const startTime = Date.now();
      
      let targetPos;
      
      if (target_type === 'coordinate' || target_type === 'block') {
        targetPos = { x: target.x, y: target.y, z: target.z };
      } else if (target_type === 'entity') {
        const entity = getEntityById(target.entity_id);
        if (!entity) {
          return {
            success: false,
            error: { code: 'NOT_FOUND', message: `实体不存在: ${target.entity_id}` },
            meta: { duration: Date.now() - startTime }
          };
        }
        targetPos = { x: entity.pos.x, y: entity.pos.y, z: entity.pos.z };
      }
      
      // 委托给执行AI
      return await executionAI.moveTo(targetPos.x, targetPos.y, targetPos.z, {
        ...options,
        targetDistance: distance,
        sprint
      });
    }
  });
  
  // ride
  ToolRegistry.register({
    name: 'ride',
    description: '骑乘实体（马、猪、船等）',
    parameterSchema: {
      type: 'object',
      properties: {
        entity_id: { type: 'string', description: '实体 ID' }
      },
      required: ['entity_id']
    },
    async invoke(params) {
      return await executionAI.ride(params.entity_id);
    }
  });
  
  // dismount
  ToolRegistry.register({
    name: 'dismount',
    description: '下马',
    parameterSchema: {
      type: 'object',
      properties: {},
      required: []
    },
    async invoke() {
      return await executionAI.dismount();
    }
  });
}
```

### 3.2 执行AI接口

```typescript
// 执行AI接口（假人控制）
interface ExecutionAI {
  // 基础移动
  moveTo(x: number, y: number, z: number, options?: MoveOptions): Promise<ResultEnvelope<MoveToData>>;
  
  // 骑乘
  ride(entity_id: string): Promise<ResultEnvelope<RideData>>;
  dismount(): Promise<ResultEnvelope<DismountData>>;
}

// 移动选项
interface MoveOptions {
  timeout?: number;
  avoidHostile?: boolean;
  allowBreak?: boolean;
  allowPlace?: boolean;
  sprint?: boolean;         // 是否疾跑
  pathfinding?: 'astar' | 'dijkstra';
  targetDistance?: number;  // 目标距离（接近实体/方块时使用）
}
```

### 3.3 执行AI实现（示例）

```typescript
class ExecutionAIImpl implements ExecutionAI {
  private player: Player;
  private pathfinder: Pathfinder;
  
  constructor(player: Player) {
    this.player = player;
    this.pathfinder = new Pathfinder();
  }
  
  async moveTo(x: number, y: number, z: number, options?: MoveOptions): Promise<ResultEnvelope<MoveToData>> {
    const startTime = Date.now();
    const startPos = this.player.pos;
    
    try {
      // 1. 计算路径
      const path = await this.pathfinder.findPath(
        startPos,
        { x, y, z },
        {
          avoidHostile: options?.avoidHostile,
          allowBreak: options?.allowBreak,
          allowPlace: options?.allowPlace
        }
      );
      
      if (!path) {
        return {
          success: false,
          error: { code: 'NO_PATH', message: '无法找到路径' },
          data: { finalPosition: startPos },
          meta: { duration: Date.now() - startTime }
        };
      }
      
      // 2. 执行移动（自动处理跳跃、攀爬、游泳等）
      const timeout = options?.timeout || 30000;
      const result = await this.executePath(path, timeout);
      
      if (!result.success) {
        return {
          success: false,
          error: { code: result.reason as any, message: '移动执行失败' },
          data: {
            finalPosition: this.player.pos,
            distance: calculateDistance(startPos, this.player.pos)
          },
          meta: {
            duration: Date.now() - startTime,
            cost: result.hungerCost ? { hunger: result.hungerCost } : undefined
          }
        };
      }
      
      // 3. 返回结果
      return {
        success: true,
        data: {
          finalPosition: this.player.pos,
          distance: calculateDistance(startPos, this.player.pos)
        },
        meta: {
          duration: Date.now() - startTime,
          cost: { hunger: result.hungerCost || 0 }
        }
      };
      
    } catch (error) {
      return {
        success: false,
        error: { code: 'INTERNAL_ERROR', message: error.message },
        data: { finalPosition: this.player.pos },
        meta: { duration: Date.now() - startTime }
      };
    }
  }
  
  private async executePath(path: Path, timeout: number): Promise<{success: boolean; reason?: string; hungerCost?: number}> {
    const startTime = Date.now();
    let hungerCost = 0;
    
    for (const node of path.nodes) {
      // 检查超时
      if (Date.now() - startTime > timeout) {
        return { success: false, reason: 'TIMEOUT', hungerCost };
      }
      
      // 移动到下一个节点（执行AI自动处理跳跃、攀爬、游泳等）
      const moveResult = await this.moveToNode(node);
      if (!moveResult.success) {
        return { success: false, reason: moveResult.reason, hungerCost };
      }
      
      // 累计饥饿消耗
      hungerCost += moveResult.hungerCost;
    }
    
    return { success: true, hungerCost };
  }
  
  private async moveToNode(node: PathNode): Promise<{success: boolean; reason?: string; hungerCost: number}> {
    // 执行AI自动处理：
    // - 走路
    // - 跳跃（遇到障碍）
    // - 攀爬（遇到梯子/藤蔓）
    // - 游泳（遇到水）
    // - 视角控制（自动看向目标）
    
    // 示例：走路
    const dx = node.x - this.player.pos.x;
    const dz = node.z - this.player.pos.z;
    const distance = Math.sqrt(dx * dx + dz * dz);
    
    // 设置朝向（自动）
    const yaw = Math.atan2(dz, dx) * 180 / Math.PI;
    this.player.setRotation(yaw, 0);
    
    // 开始移动
    this.player.setVelocity({ x: dx / distance * 0.2, y: 0, z: dz / distance * 0.2 });
    
    // 等待到达
    await this.waitForPosition(node, 1000);
    
    // 计算饥饿消耗
    const hungerCost = distance * 0.01;  // 每格 0.01 饥饿
    
    return { success: true, hungerCost };
  }
  
  async ride(entity_id: string): Promise<ResultEnvelope<RideData>> {
    const startTime = Date.now();
    const entity = getEntityById(entity_id);
    if (!entity) {
      return {
        success: false,
        error: { code: 'NOT_FOUND', message: `实体不存在: ${entity_id}` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    // 检查是否可骑乘
    if (!this.isRideable(entity)) {
      return {
        success: false,
        error: { code: 'NOT_RIDEABLE', message: '该实体不可骑乘' },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    // 检查距离
    const distance = calculateDistance(this.player.pos, entity.pos);
    if (distance > 3) {
      return {
        success: false,
        error: { code: 'TOO_FAR', message: `距离过远: ${distance.toFixed(1)}格` },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    // 执行骑乘
    this.player.ride(entity);
    
    return {
      success: true,
      data: {
        isRiding: true,
        mountType: entity.type
      },
      meta: { duration: Date.now() - startTime }
    };
  }
  
  async dismount(): Promise<ResultEnvelope<DismountData>> {
    const startTime = Date.now();
    if (!this.player.isRiding()) {
      return {
        success: false,
        error: { code: 'NOT_RIDING', message: '当前未在骑乘状态' },
        meta: { duration: Date.now() - startTime }
      };
    }
    
    this.player.dismount();
    
    return {
      success: true,
      data: { isRiding: false },
      meta: { duration: Date.now() - startTime }
    };
  }
  
  // ... 其他方法
}
```

## 5. 总结

### 5.1 核心设计

| 组件           | 职责                           |
| ------------ | ---------------------------- |
| **LLM（大脑）**  | 决定要去哪里，调用基础移动工具              |
| **移动工具**     | 封装移动能力，返回执行结果                |
| **执行AI（假人）** | 负责具体移动操作（寻路、避障、跳跃、攀爬、游泳、视角等） |

### 5.2 关键特性

1. **工具粒度适中**：一个工具 = 一个"动作"（move\_to, ride, dismount 等）
2. **有反馈**：工具返回执行结果（成功/失败/中断原因）
3. **可失败**：寻路失败、超时、被阻挡等
4. **有成本**：时间、饥饿
5. **可中断**：移动过程中可被事件中断
6. **执行AI自动处理**：跳跃、攀爬、游泳、视角控制等由执行AI自动处理

### 5.3 工具列表

| 类别   | 工具                                     |
| ---- | -------------------------------------- |
| 基础移动 | move\_to（通过 target\_type 区分坐标/实体/方块目标） |
| 骑乘   | ride, dismount                         |
