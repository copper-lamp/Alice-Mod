# 学习与记忆工具设计（Memory & Learning Tools Design）

> 版本：v1.0
> 日期：2026-07-14
> 架构：LLM 调用工具 → 执行AI 执行操作 → 返回结果
> 协议：遵循 `00-工具协议规范.md` 统一接口契约

---

## 1. 模块概述

### 1.1 目标
让 AI 能够：
- 记住玩家习惯和偏好
- 记住地图关键点（资源点、危险区域、基地）
- 记住任务经验和教训
- 记住与其他玩家的交互历史
- 通过记忆优化决策

### 1.2 架构


## 4. 工具详细设计

### 4.1 memory_store

**功能**：存储新记忆，支持单个存储和批量存储

**参数**（ParameterSchema）：

| 参数 | 类型 | 必填 | 描述 | 默认值 |
|------|------|:----:|------|:------:|
| `memory` | `object` | 否 | 单个记忆对象（与 `items` 二选一） | - |
| `memory.type` | `string` | 是 | 记忆类型：`player_habit` \| `map_point` \| `task_experience` \| `social` \| `skill` | - |
| `memory.content` | `object` | 是 | 记忆内容，根据 `type` 不同结构不同 | - |
| `memory.tags` | `array[string]` | 否 | 标签列表 | `[]` |
| `memory.importance` | `number` | 否 | 重要度 (1-10) | `5` |
| `items` | `array[object]` | 否 | 批量存储模式（与 `memory` 二选一） | - |
| `items[].type` | `string` | 是 | 记忆类型 | - |
| `items[].content` | `object` | 是 | 记忆内容 | - |
| `items[].tags` | `array[string]` | 否 | 标签列表 | `[]` |
| `items[].importance` | `number` | 否 | 重要度 (1-10) | `5` |

**返回**：`ResultEnvelope<{ memory_id?: string, memory_ids?: string[] }>`

- `memory_id`：单个存储时返回的记忆 ID
- `memory_ids`：批量存储时返回的记忆 ID 列表

**错误码**：
- `INTERNAL_ERROR`：内部错误
- `STORAGE_FULL`：存储空间已满

**示例**：
```typescript
// 存储单个记忆
const result = await memory_store({
  memory: {
    type: 'player_habit',
    content: {
      player_id: 'Steve',
      habit_type: 'preference',
      description: 'Steve 喜欢用钻石工具',
      examples: ['给我钻石镐'],
      frequency: 10,
      confidence: 0.8
    },
    tags: ['Steve', '工具偏好'],
    importance: 7
  }
});
// 返回
// {
//   success: true,
//   data: { memory_id: 'mem_1719820000' },
//   meta: { duration: 150, cost: { storage: 0.5 } }
// }

// 批量存储
const result = await memory_store({
  items: [
    {
      type: 'map_point',
      content: {
        point_type: 'resource',
        location: { x: 128, y: 20, z: -256, dimension: 'overworld' },
        name: '钻石矿脉',
        description: '大量钻石矿',
        resources: ['diamond_ore']
      },
      tags: ['钻石', '矿区'],
      importance: 9
    },
    {
      type: 'map_point',
      content: {
        point_type: 'base',
        location: { x: 0, y: 64, z: 0, dimension: 'overworld' },
        name: '主基地'
      },
      importance: 10
    }
  ]
});
// 返回
// {
//   success: true,
//   data: { memory_ids: ['mem_1719820001', 'mem_1719820002'] },
//   meta: { duration: 250, cost: { storage: 1.0 } }
// }
```

### 4.2 memory_recall

**功能**：按条件检索记忆，支持按 ID、类型、标签、文本查询和语义相似度检索

**参数**（ParameterSchema）：

| 参数 | 类型 | 必填 | 描述 | 默认值 |
|------|------|:----:|------|:------:|
| `filter` | `object` | 否 | 过滤条件 | - |
| `filter.id` | `string` | 否 | 按 ID 精确查找 | - |
| `filter.type` | `string` | 否 | 记忆类型过滤 | - |
| `filter.tags` | `array[string]` | 否 | 标签过滤（AND 逻辑） | - |
| `filter.query` | `string` | 否 | 文本查询（关键词匹配） | - |
| `filter.similar_to` | `string` | 否 | 语义相似查询（向量检索，传入记忆 ID） | - |
| `limit` | `number` | 否 | 返回数量限制 | `20` |

**返回**：`ResultEnvelope<{ memories: Memory[], total: number }>`

- `memories`：匹配的记忆列表
- `total`：匹配总数

**错误码**：
- `INTERNAL_ERROR`：内部错误
- `MEMORY_NOT_FOUND`：`filter.id` 指定的记忆不存在

**示例**：
```typescript
// 按 ID 查询
const result = await memory_recall({
  filter: { id: 'mem_123456' }
});
// 返回
// {
//   success: true,
//   data: {
//     memories: [{ type: 'player_habit', ... }],
//     total: 1
//   },
//   meta: { duration: 50 }
// }

// 按类型和标签查询
const result = await memory_recall({
  filter: {
    type: 'map_point',
    tags: ['钻石']
  },
  limit: 10
});

// 文本查询
const result = await memory_recall({
  filter: {
    query: '钻石矿位置'
  }
});

// 语义相似查询
const result = await memory_recall({
  filter: {
    similar_to: 'mem_123456',
    type: 'map_point'
  },
  limit: 5
});
```

### 4.4 memory_update

**参数**（ParameterSchema）：

| 参数 | 类型 | 必填 | 描述 | 默认值 |
|------|------|:----:|------|:------:|
| `memory_id` | `string` | 是 | 要更新的记忆 ID | - |
| `updates` | `object` | 是 | 更新内容 | - |
| `updates.content` | `object` | 否 | 更新记忆内容 | - |
| `updates.tags` | `array[string]` | 否 | 更新标签 | - |
| `updates.importance` | `number` | 否 | 更新重要度 (1-10) | - |
| `updates.increment_access` | `boolean` | 否 | 是否增加访问次数 | `false` |

**返回**：`ResultEnvelope<null>`

**错误码**：
- `INTERNAL_ERROR`：内部错误

```typescript
      confidence: 0.9

    importance: 8,
    increment_access: true
  }
});
// 返回
// {
//   success: true,
//   data: null,
//   meta: { duration: 80 }
// }
```

### 4.7 memory_manage

**功能**：记忆管理操作，支持统计、清理、导出和导入

**参数**（ParameterSchema）：

| 参数 | 类型 | 必填 | 描述 | 默认值 |
|------|------|:----:|------|:------:|
| `action` | `string` | 是 | 操作类型：`stats` \| `cleanup` \| `export` \| `import` | - |
| `params` | `object` | 否 | 操作参数 | - |
| `params.max_age` | `number` | 否 | 清理：最大年龄（秒） | `2592000`（30 天） |
| `params.min_importance` | `number` | 否 | 清理：最小重要度 | `3` |
| `params.format` | `string` | 否 | 导出：格式 `json` \| `csv` | `json` |
| `params.filter` | `object` | 否 | 导出：过滤条件 | - |
| `params.filter.type` | `string` | 否 | 导出过滤：记忆类型 | - |
| `params.filter.tags` | `array[string]` | 否 | 导出过滤：标签 | - |
| `params.data` | `any` | 否 | 导入：要导入的数据 | - |

**返回**：`ResultEnvelope<any>`

- `action='stats'` 时，`data` 包含 `{ total_count, by_type, average_importance, storage_size }`
- `action='cleanup'` 时，`data` 包含 `{ deleted_count }`
- `action='export'` 时，`data` 包含 `{ memories, format }`
- `action='import'` 时，`data` 包含 `{ imported_count }`

**错误码**：
- `INTERNAL_ERROR`：内部错误

**示例**：
```typescript
// 获取统计信息
const result = await memory_manage({ action: 'stats' });
// 返回
// {
//   success: true,
//   data: {
//     total_count: 150,
//     by_type: { player_habit: 25, map_point: 40, task_experience: 30, social: 35, skill: 20 },
//     average_importance: 6.5,
//     storage_size: '2.5 MB'
//   },
//   meta: { duration: 30 }
// }

// 清理过期记忆
const result = await memory_manage({
  action: 'cleanup',
  params: {
    max_age: 30 * 24 * 60 * 60,
    min_importance: 3
  }
});
// 返回
// {
//   success: true,
//   data: { deleted_count: 12 },
//   meta: { duration: 200 }
// }

// 导出记忆
const result = await memory_manage({
  action: 'export',
  params: {
    format: 'json',
    filter: { type: 'map_point' }
  }
});
// 返回
// {
//   success: true,
//   data: { memories: [...], format: 'json' },
//   meta: { duration: 100 }
// }

// 导入记忆
const result = await memory_manage({
  action: 'import',
  params: {
    data: { memories: [...] }
  }
});
// 返回
// {
//   success: true,
//   data: { imported_count: 50 },
//   meta: { duration: 300 }
// }
```

| 参数 | 类型 | 必填 | 描述 | 默认值 |
|------|------|:----:|------|:------:|
| `x` | `number` | 是 | 中心点 X 坐标 | - |
| `z` | `number` | 是 | 中心点 Z 坐标 | - |
| `dimension` | `string` | 是 | 维度：`overworld` \| `nether` \| `end` | - |
| `radius` | `number` | 否 | 概览半径（格） | `256` |

**返回**：`ResultEnvelope<RegionSummary>`

其中 `RegionSummary` 包含：

## 5. 记忆存储实现

### 5.1 存储架构

```typescript
class MemoryStorage {
  private memories: Map<string, Memory> = new Map();
  private vectorIndex: VectorIndex;  // 向量索引（用于语义检索）
  private storagePath: string;
  
  constructor(config: MemoryConfig) {
    this.storagePath = config.storagePath;
    this.vectorIndex = new VectorIndex({
      dimension: 768,  // 嵌入向量维度
      metric: 'cosine'
    });
    this.loadFromDisk();
  }
  
  // 存储记忆
  async store(memory: Memory): Promise<string> {
    const memoryId = this.generateId();
    memory.id = memoryId;
    memory.created_at = Date.now();
    memory.updated_at = Date.now();
    memory.access_count = 0;
    
    this.memories.set(memoryId, memory);
    
    // 添加到向量索引
    const embedding = await this.generateEmbedding(memory);
    this.vectorIndex.add(memoryId, embedding);
    
    // 持久化
    await this.saveToDisk();
    
    return memoryId;
  }
  
  // 检索记忆
  async recall(filter: MemoryFilter): Promise<Memory[]> {
    let results = Array.from(this.memories.values());
    
    // 应用过滤条件
    if (filter.type) {
      results = results.filter(m => m.type === filter.type);
    }
    if (filter.tags && filter.tags.length > 0) {
      results = results.filter(m => 
        filter.tags!.every(tag => m.tags.includes(tag))
      );
    }
    if (filter.importance_min) {
      results = results.filter(m => m.importance >= filter.importance_min!);
    }
    if (filter.created_after) {
      results = results.filter(m => m.created_at >= filter.created_after!);
    }
    
    // 排序
    if (filter.sort_by) {
      results.sort((a, b) => {
        switch (filter.sort_by) {
          case 'importance':
            return b.importance - a.importance;
          case 'created_at':
            return b.created_at - a.created_at;
          case 'access_count':
            return b.access_count - a.access_count;
          case 'updated_at':
            return b.updated_at - a.updated_at;
          default:
            return 0;
        }
      });
    }
    
    // 限制数量
    if (filter.limit) {
      results = results.slice(0, filter.limit);
    }
    
    return results;
  }
  
  // 语义检索
  async getSimilar(query: string, options: SimilarOptions): Promise<SimilarResult[]> {
    const queryEmbedding = await this.generateEmbedding({ content: query });
    const similarIds = this.vectorIndex.search(queryEmbedding, options.limit || 5);
    
    const results: SimilarResult[] = [];
    for (const { id, similarity } of similarIds) {
      if (similarity < (options.similarity_threshold || 0.7)) continue;
      
      const memory = this.memories.get(id)!;
      if (options.type && memory.type !== options.type) continue;
      
      results.push({ memory, similarity });
    }
    
    return results.sort((a, b) => b.similarity - a.similarity);
  }
  
  // 生成嵌入向量
  private async generateEmbedding(memory: Memory | { content: any }): Promise<number[]> {
    // 将记忆内容转换为文本
    const text = this.memoryToText(memory);
    
    // 调用嵌入模型（如 OpenAI embeddings）
    const embedding = await openai.embeddings.create({
      model: 'text-embedding-3-small',
      input: text
    });
    
    return embedding.data[0].embedding;
  }
  
  // 记忆转文本
  private memoryToText(memory: Memory | { content: any }): string {
    if ('type' in memory) {
      // 完整记忆对象
      switch (memory.type) {
        case 'player_habit':
          return `${memory.content.player_id} ${memory.content.description}`;
        case 'map_point':
          return `${memory.content.name} ${memory.content.description} 位置 ${memory.content.location.x},${memory.content.location.y},${memory.content.location.z}`;
        case 'task_experience':
          return `${memory.content.description} ${memory.content.lessons.join(' ')} ${memory.content.tips.join(' ')}`;
        case 'social':
          return `${memory.content.player_id} ${memory.content.notes.join(' ')}`;
        case 'skill':
          return `${memory.content.description} ${memory.content.code || ''}`;
        default:
          return JSON.stringify(memory.content);
      }
    } else {
      // 查询文本
      return memory.content;
    }
  }
  
  // 持久化到磁盘
  private async saveToDisk(): Promise<void> {
    const data = {
      memories: Array.from(this.memories.values()),
      vectorIndex: this.vectorIndex.serialize()
    };
    
    await fs.writeFile(
      this.storagePath,
      JSON.stringify(data, null, 2)
    );
  }
  
  // 从磁盘加载
  private async loadFromDisk(): Promise<void> {
    if (!fs.existsSync(this.storagePath)) return;
    
    const data = JSON.parse(await fs.readFile(this.storagePath, 'utf-8'));
    
    // 加载记忆
    for (const memory of data.memories) {
      this.memories.set(memory.id, memory);
    }
    
    // 加载向量索引
    this.vectorIndex.deserialize(data.vectorIndex);
  }
}
```
