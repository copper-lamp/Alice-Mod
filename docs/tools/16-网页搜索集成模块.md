# 网页搜索集成模块

> 版本：v1.0
> 日期：2026-07-11
> 状态：开发中

---

## 1. 概述

将 [free-search-mcp](https://github.com/sweetcornna/free-search-mcp) 的核心搜索能力以**原生 DuckDuckGo API 封装**的方式集成到 Agent Core 中。

### 1.1 设计原则

- **零外部依赖**：不依赖 MCP 协议、不依赖 Python/Playwright 运行时
- **零 API Key**：使用 DuckDuckGo 的公开 API（Instant Answer API + HTML fallback）
- **模块化**：独立模块 `src/main/search/`，与 Agent Core 其他模块松耦合
- **精简工具**：只暴露 2 个核心工具

### 1.2 与 free-search-mcp 的区别

| 特性 | free-search-mcp | 本模块 |
|------|----------------|--------|
| 运行时 | Python 3.11+ | Node.js（内置） |
| 搜索引擎 | 多引擎（DDG/Bing/Google/百度等） | DuckDuckGo（免费、无需 Key） |
| 浏览器渲染 | Playwright | 无（纯 HTTP） |
| 工具数量 | 9 个 | 2 个（精简） |
| 安装部署 | clone + uv sync | 零安装 |

## 2. 工具定义

### 2.1 `web_search`

搜索互联网。

| 属性 | 值 |
|------|-----|
| **名称** | `web_search` |
| **分类** | `knowledge` |
| **描述** | 搜索互联网，查找实时信息。支持 LLM 知识截止日期之后的查询 |
| **参数** | `query`（必填）, `limit`（可选，默认 5） |
| **输出** | Markdown 编号列表或 JSON |

### 2.2 `web_fetch`

获取指定 URL 的网页内容。

| 属性 | 值 |
|------|-----|
| **名称** | `web_fetch` |
| **分类** | `knowledge` |
| **描述** | 获取网页正文内容。适用于阅读搜索结果中的具体文章 |
| **参数** | `url`（必填） |
| **输出** | Markdown（标题 + 正文）或 JSON |

---

## 3. 搜索架构

```
web_search / web_fetch
       │
       ▼
  SearchClient
       │
       ├── 1. DuckDuckGo Instant Answer API (JSON)
       │      https://api.duckduckgo.com/?q=...&format=json
       │
       └── 2. Fallback: DuckDuckGo HTML 搜索
              https://html.duckduckgo.com/html/?q=...
```
